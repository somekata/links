INTRODUCTION

統計学とは何か
── なぜ必要か

統計学の目的

統計学とは、データから情報を引き出し、不確実性を定量的に扱う学問です。現実の世界では、どれだけ注意深く実験や調査を行っても「ばらつき」が存在します。統計学は、このばらつきを無視するのではなく、正面から数値として扱うための道具立てを提供します。

生物学・医学・疫学の研究では、統計学は「実験結果が偶然によるものか、それとも本当の効果があるのか」を判断するための共通言語です。論文を読む際にも、自分で研究を行う際にも、統計の基礎知識は不可欠です。

📌 統計学の二大目的

記述統計（Descriptive Statistics）：データの特徴を要約・整理する。平均、標準偏差、グラフなど。

推測統計（Inferential Statistics）：標本（手元のデータ）から母集団（全体）について推論する。検定、信頼区間など。

母集団と標本

母集団（Population）とは、興味の対象となるすべての個体・測定値の集合です。たとえば「日本人成人男性の血圧」を研究したい場合、母集団は日本人成人男性全員の血圧値です。

実際には母集団全員を測定することは不可能なので、標本（Sample）と呼ばれる一部のデータを取得し、そこから母集団の性質を推定します。

母集団と標本の関係
📖 用語整理

母集団の平均を μ（ミュー）、標準偏差を σ（シグマ） と表記します（ギリシャ文字＝真の値）。

標本の平均を x̄（エックスバー）、標準偏差を s と表記します（ローマ字＝推定値）。

DATA TYPES

データの種類と分布

データの測定尺度

統計解析の第一歩は、手元のデータがどの種類に属するかを見極めることです。データの種類によって、使用できる統計手法が決まります。

尺度 特徴 例（生物・医学） 使える統計
名義尺度
Nominal		 分類のみ。順序なし 血液型、遺伝子型、性別、疾患の有無 頻度、χ²検定
順序尺度
Ordinal		 順序あり。間隔は不均等 病期（Stage I〜IV）、重症度スコア、Likert尺度 中央値、Mann-Whitney検定
間隔尺度
Interval		 等間隔。絶対ゼロなし 体温（℃）、暦年、pH 平均、t検定
比率尺度
Ratio		 等間隔＋絶対ゼロあり 体重、身長、血圧、細胞数、濃度 平均、SD、全パラメトリック検定

量的変数 vs 質的変数

実務上は、量的変数（連続・離散）と質的変数（カテゴリカル）に大別して考えると便利です。

🔬 生物学・医学での例

量的連続変数：血糖値（mg/dL）、腫瘍サイズ（mm）、タンパク質発現量（相対値）

量的離散変数：コロニー数、変異数、転移個数

質的変数：喫煙の有無（Yes/No）、投薬群 vs 対照群、がんの組織型

DESCRIPTIVE STATISTICS

記述統計
── 平均・中央値・最頻値

代表値（中心傾向の指標）

データの「中心」を一つの数値で表すものを代表値と言います。最もよく使われる三種類を理解しましょう。

① 算術平均（Mean）

すべての値を足して個数で割った値。最もポピュラーですが、外れ値（outlier）に敏感という弱点があります。

FORMULA ── 算術平均
x̄ = (x₁ + x₂ + ⋯ + xₙ) / n = Σxᵢ / n
x₁, x₂, …, xₙ：各データ値　　n：データ数

② 中央値（Median）

データを小さい順に並べたときの中央の値。外れ値の影響を受けにくいため、分布が歪んでいるデータ（生存期間、収入、サイトカイン濃度など）に適しています。

③ 最頻値（Mode）

最も頻繁に出現する値。カテゴリカルデータに有用です。

⚠️ 平均だけに頼らない

右に裾が長い分布（例：血清CRP値、生存期間）では、平均が中央値より大きく上振れします。このようなデータでは中央値と四分位範囲（IQR）を使うか、対数変換を検討します。

5人の患者の血清IL-6値（pg/mL）：3, 5, 6, 7, 120

平均 = (3+5+6+7+120) / 5 = 28.2　（120の影響で高い）

中央値 = 6　（外れ値120に左右されない）

→ この分布では中央値の方がデータの典型例をよく表す。

DISPERSION

ばらつきの指標
── 分散・SD・四分位範囲

なぜばらつきを測るか

代表値だけでは不十分です。平均が同じでも、データが密集している場合と広く散らばっている場合では、まったく異なる意味を持ちます。ばらつき（分散）の指標は、データの信頼性・精度・個人差を表す重要な情報です。

分散と標準偏差（SD）

FORMULA ── 標本分散（s²）と標準偏差（s）
s² = Σ(xᵢ − x̄)² / (n − 1)
SD = s = √[ Σ(xᵢ − x̄)² / (n − 1) ]
分母が n ではなく (n−1) なのは、標本から母分散を不偏推定するためです（ベッセルの補正）。

標準偏差（SD：Standard Deviation）は、「典型的な1個のデータ値が平均からどれくらい離れているか」を表します。単位は元のデータと同じなので直感的に解釈できます。

📌 SD の直感的意味

成人男性の身長：平均 170 cm、SD = 6 cm とすると、多くの人（約68%）は 164〜176 cm の範囲に収まることを意味します。

四分位範囲（IQR）

データを4等分したときの第1四分位数（Q1, 下位25%点）と第3四分位数（Q3, 上位25%点）の差。IQR = Q3 − Q1

中央値と組み合わせて使用し、外れ値の影響を受けません。歪んだ分布に適しています。

指標 使う場面 表記例（論文）
平均 ± SD 正規分布に近いデータ（体重、身長、血圧） 170.2 ± 6.1 cm
中央値 [IQR] 歪んだ分布（生存期間、サイトカイン、細菌数） 6.0 [3.5–18.2] pg/mL
中央値 (範囲) 小標本・外れ値がある場合 6.0 (3–120) pg/mL

MOST IMPORTANT CONCEPT

SD と SE の違い
── 最重要ポイント

概念の違い：何のばらつきか

📌 一行で言えば

SD（標準偏差）：個々のデータが平均からどれだけばらついているか → データのばらつき

SE（標準誤差）：標本平均が母平均からどれだけずれうるか → 推定値（平均）の精度

同じ実験を何度も繰り返し、毎回標本平均 x̄ を計算したとします。その x̄ 自身にもばらつきがあります。SE はこの「平均のばらつき」を表します。

FORMULA ── 標準誤差（SE / SEM）
SE = SD / √n
SD：標本標準偏差　　n：標本サイズ
標本数が増えると SE は小さくなる（推定が精確になる）。SD は変わらない。

数値例で理解する

10人の患者の収縮期血圧を測定：

値（mmHg）：128, 135, 122, 140, 118, 132, 129, 125, 138, 133

x̄（平均） = 130.0 mmHg

SD = 6.7 mmHg　→「患者個人の血圧は平均から約6.7 mmHgばらつく」

SE = 6.7 / √10 = 2.1 mmHg　→「この平均130 mmHgという推定値自体の精度は±2.1 mmHg程度」

SD と SE の視覚的比較（n を増やすと）

どちらを使うべきか

使う場面 適切な指標 理由
データのばらつき・個人差を示したい SD 集団内の変動を表すため
平均値の推定精度を示したい SE 「この平均はどれくらい信頼できるか」を示すため
臨床的基準値・正常範囲を示したい SD（平均±2SD） 個人の測定値が正常範囲内かを判断するため
グラフのエラーバー（比較目的） SE または 95%CI 群間比較の精度を視覚化するため
論文の基本集計表 正規分布→SD、非正規→IQR 分布の形に応じて使い分ける
⚠️ よくある誤用

SE は SD より必ず小さい（n > 1 の場合）ため、グラフを「小さく見せたい」目的で SE を使う誤用が存在します。SE を使う場合は、それが「平均の精度」を示しているものであることを明示してください。

信頼区間（95% CI）は SE の約2倍として計算でき、SE よりも解釈しやすいため、現代の論文では CI を好む傾向があります。

NORMAL DISTRIBUTION

正規分布と標準化

正規分布（ガウス分布）

多くの自然現象・生物学的測定値は、平均を中心に左右対称の釣り鐘型をした正規分布（Normal Distribution）に従います。統計検定の多くが正規分布を前提とするため、基礎として必須の概念です。

正規分布と SD の関係
📌 正規分布の 68–95–99.7 ルール

平均 ± 1SD の範囲に 約68%、± 2SD に 約95%、± 3SD に 約99.7% のデータが含まれます。

これを利用して「平均 ± 2SD」を正常範囲の目安とすることが多いです（外れる約5%が「異常値」）。

標準化と Z スコア

異なる単位・異なるスケールのデータを比較するために、Z スコア（標準スコア）に変換します。

FORMULA ── Z スコア
z = (x − μ) / σ
z スコアは「平均から何 SD 分離れているか」を表す無次元の数値。平均=0、SD=1 の標準正規分布に従う。

患者A：BMI = 28（集団平均 = 23, SD = 4）　→　z = (28−23)/4 = 1.25

患者B：収縮期血圧 = 145 mmHg（平均 = 120, SD = 15）　→　z = (145−120)/15 = 1.67

→ BMI より血圧の方が「平均からの乖離」が大きいことが一目でわかる。

BIOLOGICAL STATISTICS 生物統計

生物統計の基礎

生物学データの特徴

生物学実験データは統計的に扱いにくい性質を持つことが多くあります。これを理解することが適切な解析の第一歩です。

📖 生物学データの典型的な特徴

小標本：n = 3〜10 程度が多い。統計的検出力が低い。

非正規分布：遺伝子発現量、細菌コロニー数、IC50値などは対数正規分布に従うことが多い。

外れ値：生物学的変動が大きく、個体差が顕著。

技術的反復 vs 生物学的反復：重要な区別（後述）。

生物学的反復 vs 技術的反復

生物学実験における最重要概念の一つです。

生物学的反復（Biological Replicate） 技術的反復（Technical Replicate）
定義 独立した生物学的サンプル（異なる個体・細胞継代・マウス） 同一サンプルを複数回測定
目的 生物学的変動を捉える。統計的推論に使う 測定誤差を評価・低減する
n として計上 ✅ YES ❌ NO（平均して1点に）
例 3匹の異なるマウスから採取した肝臓サンプル（n=3） 同一マウスの肝臓から3ウェルで測定した吸光度
⚠️ 擬似反復（Pseudoreplication）

技術的反復を生物学的反復として n に計上する誤りを擬似反復（pseudoreplication）と言います。これは標本サイズを水増しし、p 値を不当に小さくするため、論文で非常に問題になります。

対数変換と対数正規分布

遺伝子発現量（RPKM、TPM）、抗体希釈系列、EC50・IC50、細菌増殖などは、対数正規分布（log-normal distribution）に従うことが多く、対数変換後に正規分布の統計手法を適用します。

APPROACH ── 対数変換の手順
1. 元データ x に対して log₁₀(x) または ln(x) を計算
2. 変換後データの正規性を確認（Shapiro-Wilk 検定など）
3. 変換後データで t 検定・ANOVA を実施
4. 結果は幾何平均（Geometric Mean）で報告
幾何平均 = 10^（対数変換後の算術平均）

生物学でよく使う基本的な検定

場面 正規分布が仮定できる場合 できない場合（ノンパラメトリック）
2群の比較 対応なし t 検定
対応あり t 検定（paired）		 Mann-Whitney U 検定
Wilcoxon 符号順位検定
3群以上の比較 一元配置 ANOVA Kruskal-Wallis 検定
カテゴリカルデータ χ²（カイ二乗）検定、Fisher の正確検定

EPIDEMIOLOGICAL STATISTICS 疫学統計

疫学統計の基礎

疫学の基本指標

疫学（Epidemiology）は、集団における疾患の頻度・分布・決定要因を研究する学問です。統計指標が臨床意思決定や公衆衛生政策の根拠となります。

罹患率・有病率

FORMULA ── 疾病頻度の基本指標
有病率（Prevalence） = 疾患あり人数 / 総人口
罹患率（Incidence Rate） = 新規発症数 / 観察人時
有病率：ある時点での疾患保有割合（断面）
罹患率：一定期間に新たに発症する速度（縦断）

リスク比・オッズ比・ハザード比

疫学では「曝露によってどれだけ疾患リスクが変わるか」を定量化します。

FORMULA ── 疫学の主要効果指標
リスク比（RR） = 曝露群のリスク / 非曝露群のリスク
オッズ比（OR） = (a/b) / (c/d) = ad / bc
ハザード比（HR） = 曝露群のハザード / 非曝露群のハザード
2×2分割表：a=曝露＋疾患あり、b=曝露＋疾患なし、c=非曝露＋疾患あり、d=非曝露＋疾患なし
指標 主な用途 RR=1との関係 使われる研究デザイン
リスク比（RR） コホート研究・RCT RR=1：関連なし
RR>1：リスク増加		 前向きコホート、RCT
オッズ比（OR） 稀な疾患・後ろ向き研究 OR=1：関連なし
疾患が稀なときRR≒OR		 症例対照研究、ロジスティック回帰
ハザード比（HR） 時間依存的アウトカム HR=1：差なし 生存分析（Cox回帰）

読み方の例：「喫煙者の肺がんリスク比 RR = 15.0（95%CI: 12.0–18.5）」

→ 喫煙者は非喫煙者に比べ肺がんになるリスクが15倍。信頼区間が1をまたがないため統計的に有意。

感度・特異度・陽性的中率

診断検査や検診の性能評価に使われる指標群です。

指標 定義 意味
感度（Sensitivity） TP / (TP + FN) 本当に病気の人を正しく陽性と判定できる割合
特異度（Specificity） TN / (TN + FP) 本当に健康な人を正しく陰性と判定できる割合
陽性的中率（PPV） TP / (TP + FP) 陽性と出た中で本当に病気の人の割合（有病率に依存）
陰性的中率（NPV） TN / (TN + FN) 陰性と出た中で本当に健康な人の割合
⚠️ 有病率の罠（ベイズの定理）

感度 99%・特異度 99% の検査でも、疾患有病率が 0.1% の場合、陽性的中率は約9%にすぎません。稀な疾患のスクリーニングでは偽陽性が多く出るため、有病率を考慮した解釈が重要です。

交絡因子とバイアス

疫学研究では、因果関係の推論を歪める要因を理解することが重要です。

📖 主要な概念

交絡因子（Confounding factor）：曝露とアウトカムの両方に関連し、見かけ上の関連を生む第三の変数。例：コーヒー飲用と肺がん（喫煙が交絡）。

選択バイアス（Selection bias）：研究参加者が母集団を代表していないことによる歪み。

情報バイアス（Information bias）：測定・記録・想起の誤りによる歪み。想起バイアス、検査者バイアスなど。

HYPOTHESIS TESTING

統計的検定の入口

仮説検定の論理

統計的仮説検定は、「偶然でこのような結果が得られる確率はどれくらいか」を定量化することで、研究の発見を評価します。

📌 検定の基本フロー

① 帰無仮説（H₀）を立てる：「差はない」「効果はない」（反証したい仮説）

② 対立仮説（H₁）を立てる：「差がある」（証明したい仮説）

③ 検定統計量を計算する：t 値、χ²値など

④ p 値を求める：H₀ が真のときこのデータが得られる確率

⑤ 有意水準 α と比較して判断する：慣例的に α = 0.05

p 値の正しい解釈

p 値は最も誤解されやすい統計概念の一つです。

⚠️ p 値に関する誤解

❌「p < 0.05 は効果が大きいことを意味する」→ 誤り。効果の大きさ（効果量）とは別の概念です。

❌「p = 0.03 は帰無仮説が真である確率が3%」→ 誤り。p 値は帰無仮説の下でデータが得られる確率です。

✅ p 値は「帰無仮説が正しいとした場合に、観察された（またはそれ以上の）差が生じる確率」です。

第一種・第二種の過誤と検出力

H₀ が真（差なし） H₀ が偽（差あり）
H₀ を棄却（p < 0.05） 第一種の過誤（α）
偽陽性。慣例 α = 0.05		 正しい判断 ✅
検出力（Power） = 1−β
H₀ を保留（p ≥ 0.05） 正しい判断 ✅ 第二種の過誤（β）
偽陰性。慣例 β = 0.20

検出力（Power）= 1 − βは「真に差があるときに、正しくそれを検出できる確率」です。サンプルサイズが大きいほど検出力は上がります。研究計画の段階でサンプルサイズ計算（検出力計算）を行うことが必須です。

信頼区間（Confidence Interval）

95% 信頼区間とは、「同じ手順で実験を繰り返した場合、95%の確率でこの区間が真の値を含む」という意味です（「真の値が95%の確率でこの区間にある」という意味ではありません）。

FORMULA ── 95% 信頼区間（正規近似）
95% CI = x̄ ± 1.96 × SE = x̄ ± 1.96 × (SD / √n)
1.96 は標準正規分布の 97.5 パーセンタイル値
📌 CI と p 値の対応

2群の比較で、差の 95% CI がゼロをまたがない ⟺ p < 0.05 で有意差あり（と対応します）。

現代の統計報告では、p 値単独より 効果量 + 信頼区間 の報告が推奨されています。

KEY TAKEAWAYS

基本編 総まとめ

✅ 基本編で押さえるべき10のポイント
  1. 統計学は「不確実性を定量的に扱う」道具である
  2. データには名義・順序・間隔・比率の4種の尺度がある
  3. 代表値（平均・中央値）は分布の形に応じて使い分ける
  4. SD = データのばらつき、SE = 平均値の精度（最重要！）
  5. SE は SD を √n で割ったもの。n が増えると SE は小さくなるが SD は変わらない
  6. 正規分布では平均±2SD に約95%のデータが含まれる
  7. 生物学では生物学的反復のみが統計的 n として有効（擬似反復に注意）
  8. 疫学の効果指標：RR（コホート）、OR（症例対照）、HR（生存分析）
  9. p < 0.05 は「偶然でないことの証拠」であり、効果の大きさではない
  10. 効果量と信頼区間を p 値とセットで報告するのが現代の標準
