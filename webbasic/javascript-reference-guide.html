JavaScript かんたんガイド

よく使うJavaScriptの構文・メソッドを、シンプルな言葉でまとめました

// 変数の宣言と関数の基本
const message = "こんにちは";
function greet(name) {
  return message + "、" + name;
}
console.log(greet("世界")); // → "こんにちは、世界"
基本構文 型・変数 配列 文字列 オブジェクト 非同期 DOM操作
📝

基本構文 — 制御フローと関数

JavaScriptの基本的な書き方です。条件分岐・繰り返し・関数を組み合わせてプログラムを構築します。

if / else
条件分岐
条件が true のとき if ブロックを、false のとき else ブロックを実行します。
if (score >= 80) {
  console.log("合格");
} else {
  console.log("不合格");
}
for
for ループ
決まった回数だけ処理を繰り返します。「初期化; 条件; 更新」の3つをセミコロンで区切って書きます。
for (let i = 0; i < 3; i++) {
  console.log(i); // 0, 1, 2
}
for...of
for...of ループ
配列・文字列などの反復可能オブジェクトの要素を1つずつ取り出します。インデックスが不要なときに便利です。
const fruits = ["りんご", "バナナ", "みかん"];
for (const f of fruits) {
  console.log(f);
}
function
関数宣言
処理をまとめて名前を付けます。return で値を返し、引数で値を受け取ります。巻き上げ（ホイスティング）により宣言前でも呼び出し可能です。
function add(a, b) {
  return a + b;
}
console.log(add(3, 4)); // → 7
アロー関数
アロー関数（=>）
=> を使ったコンパクトな関数記法です。処理が1行なら return と {} を省略できます。コールバックでよく使われます。
const double = (n) => n * 2;
console.log(double(5)); // → 10

const greet = (name) => {
  return `こんにちは、${name}`;
};
try / catch
例外処理
エラーが起きそうなコードを try で囲み、エラー発生時の処理を catch に書きます。プログラムのクラッシュを防げます。
try {
  const data = JSON.parse(invalidJson);
} catch (err) {
  console.error("エラー:", err.message);
}
📦

型・変数 — データの種類と宣言

JavaScriptには動的型付けがあり、変数に様々な型の値を格納できます。const・let・varの使い分けが重要です。

const / let
変数宣言
基本は const（再代入不可）を使い、後で値を変える必要があるときだけ let を使います。var はスコープが広すぎるので現代では非推奨です。
const name = "田中"; // 再代入不可
let count = 0; // 再代入可
count = 1; // OK
// name = "鈴木"; // エラー！
typeof
型の確認
値の型を文字列で返します。"string"・"number"・"boolean"・"object"・"undefined"・"function" などがあります。
typeof "hello" // → "string"
typeof 42 // → "number"
typeof true // → "boolean"
typeof [] // → "object" ※注意
💡 注意：typeof null も "object" を返します。これはJavaScriptの歴史的バグです。
テンプレート文字列
テンプレートリテラル（バッククォート）
バッククォート（`）で囲むと、${変数} の形で文字列の中に変数を埋め込めます。改行もそのまま書けます。
const name = "田中";
const age = 30;
console.log(`${name}さんは${age}歳です`);
// → "田中さんは30歳です"
分割代入
分割代入（Destructuring）
配列やオブジェクトから値を取り出して変数に一括代入できます。コードを簡潔に書けます。
// 配列の分割代入
const [first, second] = [10, 20];

// オブジェクトの分割代入
const { name, age } = { name: "田中", age: 30 };
console.log(name); // → "田中"
スプレッド構文
スプレッド演算子（...）
配列やオブジェクトを展開します。コピーや結合を簡単に書けます。
const a = [1, 2];
const b = [...a, 3, 4]; // → [1, 2, 3, 4]

const obj1 = { x: 1 };
const obj2 = { ...obj1, y: 2 }; // → { x: 1, y: 2 }
🗂️

配列 — リストデータの操作

配列はデータのリストを扱う最も基本的な構造です。map・filter・reduce などのメソッドをマスターすると、データ操作が格段に楽になります。

push / pop
末尾への追加・削除
push() は配列の末尾に要素を追加し、pop() は末尾の要素を取り出して返します。元の配列を変更します。
const arr = [1, 2, 3];
arr.push(4); // → [1, 2, 3, 4]
arr.pop(); // → 4 を返し、配列は [1, 2, 3]
map()
要素を変換して新しい配列を作る
各要素にコールバックを適用し、その戻り値で新しい配列を作ります。元の配列は変更しません。
const nums = [1, 2, 3];
const doubled = nums.map(n => n * 2);
// → [2, 4, 6]
filter()
条件に合う要素だけ抽出
コールバックが true を返した要素だけを集めた新しい配列を返します。
const nums = [1, 2, 3, 4, 5];
const evens = nums.filter(n => n % 2 === 0);
// → [2, 4]
find() / findIndex()
条件に合う最初の要素を探す
find() は条件を満たす最初の要素を、findIndex() はそのインデックスを返します。見つからない場合は undefined / -1 を返します。
const users = [{id: 1, name: "田中"}, {id: 2, name: "鈴木"}];
const user = users.find(u => u.id === 2);
// → { id: 2, name: "鈴木" }
reduce()
配列を1つの値にまとめる
各要素を「累積値」に畳み込んで最終的に1つの値を返します。合計・最大値・グループ化など幅広く使えます。
const nums = [1, 2, 3, 4];
const sum = nums.reduce((acc, n) => acc + n, 0);
// → 10
includes() / some() / every()
存在・条件の確認
includes() は値の存在確認、some() は1つでも条件を満たすか、every() は全要素が条件を満たすかを確認します。すべて true/false を返します。
[1, 2, 3].includes(2) // → true
[1, 2, 3].some(n => n > 2) // → true
[1, 2, 3].every(n => n > 0) // → true
🔤

文字列 — テキストの操作

文字列はJavaScriptでよく扱うデータ型です。検索・切り出し・変換など便利なメソッドが多数あります。

slice()
文字列の切り出し
開始インデックスから終了インデックス（含まない）までを切り出します。負の値を使うと末尾からカウントします。
const s = "Hello, World!";
s.slice(0, 5) // → "Hello"
s.slice(-6) // → "World!"
includes() / startsWith() / endsWith()
文字列の検索
文字列が特定の文字列を含むか、特定の文字列で始まる・終わるかを確認します。
"JavaScript".includes("Script") // → true
"JavaScript".startsWith("Java") // → true
"JavaScript".endsWith("Script") // → true
replace() / replaceAll()
文字列の置換
replace() は最初の一致だけ、replaceAll() はすべての一致を置換します。正規表現も使えます。
"aa-bb-cc".replace("-", "/") // → "aa/bb-cc"
"aa-bb-cc".replaceAll("-", "/") // → "aa/bb/cc"
split() / join()
分割・結合
split() は文字列を区切り文字で分割して配列に、join() は配列を区切り文字でつないで文字列にします。セットで覚えましょう。
"a,b,c".split(",") // → ["a","b","c"]
["a", "b", "c"].join("-") // → "a-b-c"
trim() / padStart() / padEnd()
空白除去・文字埋め
trim() は両端の空白を取り除きます。padStart() / padEnd() は指定の長さになるまで文字を埋めます。
" hello ".trim() // → "hello"
"5".padStart(3, "0") // → "005"
"ok".padEnd(5, ".") // → "ok..."
🗃️

オブジェクト — キーと値のペア

オブジェクトは名前（キー）と値のペアでデータを管理します。Object の静的メソッドを使うとデータ変換が楽になります。

Object.keys() / values() / entries()
キー・値・ペアの配列を取得
オブジェクトのキー・値・[キー, 値] ペアの配列を返します。map や for...of と組み合わせてよく使います。
const obj = { a: 1, b: 2, c: 3 };
Object.keys(obj) // → ["a", "b", "c"]
Object.values(obj) // → [1, 2, 3]
Object.entries(obj) // → [["a",1],["b",2],["c",3]]
Object.assign()
オブジェクトのコピー・結合
1つ以上のオブジェクトのプロパティをターゲットにコピーします。スプレッド構文（{ ...obj }）の方が読みやすい場合も多いです。
const target = { a: 1 };
Object.assign(target, { b: 2 }, { c: 3 });
// target → { a: 1, b: 2, c: 3 }
JSON.stringify() / JSON.parse()
JSON変換
stringify() はオブジェクトをJSON文字列に変換し、parse() はJSON文字列をオブジェクトに戻します。APIやデータ保存でよく使います。
const obj = { name: "田中", age: 30 };
const json = JSON.stringify(obj);
// → '{"name":"田中","age":30}'
JSON.parse(json).name // → "田中"
オプショナルチェーン (?.)
安全なプロパティアクセス
プロパティが null / undefined のときにエラーではなく undefined を返します。深いネストのデータを扱うときに便利です。
const user = { profile: { name: "田中" } };
user?.profile?.name // → "田中"
user?.address?.city // → undefined（エラーなし）
Null合体演算子 (??)
デフォルト値の設定
左辺が null または undefined のとき右辺を返します。0 や空文字 "" は有効な値として扱います（|| との違い）。
const name = null ?? "ゲスト"; // → "ゲスト"
const count = 0 ?? 10; // → 0 (0は有効)
const cnt2 = 0 || 10; // → 10 (|| は0をfalsy扱い)
⏳

非同期処理 — Promise と async/await

JavaScriptは非同期処理を多用します。APIからデータを取得したりファイルを読み込む際に必須の知識です。

Promise
Promise — 非同期処理の結果
非同期操作の「将来の結果」を表すオブジェクトです。成功（then）・失敗（catch）・完了（finally）で処理を連鎖できます。
fetch("https://api.example.com/data")
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));
async / await
async/await — 同期的に書く非同期処理
Promise を await で「待つ」ことで、非同期処理を同期的なコードのように書けます。async の付いた関数の中でだけ使えます。
async function getData() {
  try {
    const res = await fetch("/api/data");
    const data = await res.json();
    return data;
  } catch (err) {
    console.error(err);
  }
}
💡 ベストプラクティス： await は必ず try...catch と組み合わせてエラー処理をしましょう。
Promise.all()
複数の非同期処理を並行実行
複数のPromiseを同時に実行し、すべて完了したら結果の配列を返します。順番に待つより高速です。1つでも失敗するとrejectされます。
const [users, posts] = await Promise.all([
  fetch("/api/users").then(r => r.json()),
  fetch("/api/posts").then(r => r.json()),
]);
setTimeout / setInterval
タイマー
setTimeout() は指定時間後に1回だけ実行し、setInterval() は指定間隔で繰り返し実行します。clearInterval() で停止できます。
setTimeout(() => console.log("1秒後"), 1000);

const id = setInterval(() => {
  console.log("1秒ごと");
}, 1000);
clearInterval(id); // 停止

DOM操作 — ページの要素を操作する

DOM（Document Object Model）を通じてHTMLの要素を取得・変更・追加できます。ブラウザ上でのインタラクティブな処理の基本です。

querySelector()
要素の取得
CSSセレクターで最初の一致要素を取得します。querySelectorAll() はすべての一致要素をNodeListで返します。
const btn = document.querySelector("#myBtn");
const items = document.querySelectorAll(".item");
items.forEach(el => console.log(el.textContent));
addEventListener()
イベントリスナーの追加
クリック・入力・スクロールなどのイベントを監視し、発生時にコールバックを実行します。ユーザーインタラクションの基本です。
const btn = document.querySelector("button");
btn.addEventListener("click", (e) => {
  console.log("クリックされました");
  e.preventDefault(); // デフォルト動作を止める
});
classList
クラスの操作
add()・remove()・toggle()・contains() でCSSクラスを動的に操作します。スタイルの切り替えに多用します。
const el = document.querySelector(".box");
el.classList.add("active");
el.classList.remove("hidden");
el.classList.toggle("open"); // あれば削除、なければ追加
innerHTML / textContent
コンテンツの書き換え
textContent はテキストのみを安全に書き換えます。innerHTML はHTMLタグごと書き換えますが、ユーザー入力をそのまま渡すとXSSの危険があります。
const el = document.querySelector("#output");
el.textContent = "安全なテキスト";
el.innerHTML = "<strong>HTML込み</strong>";
💡 セキュリティ：ユーザー入力を innerHTML にそのまま入れると XSS 攻撃に脆弱になります。ユーザーデータには必ず textContent を使いましょう。
createElement / appendChild
要素の動的生成・追加
createElement() で新しいHTML要素を作り、appendChild() や append() でDOMに追加します。リストの動的生成などに使います。
const li = document.createElement("li");
li.textContent = "新しいアイテム";
li.classList.add("item");
document.querySelector("ul").appendChild(li);
