HTML要素 かんたんガイド

よく使うHTML要素を、シンプルな言葉でまとめました

文書情報 構造・区分 テキスト インライン 画像・メディア フォーム 表
📄

文書情報（メタデータ）

ページの「裏側」にある情報を定義する要素です。ブラウザや検索エンジンが読みますが、画面には直接表示されません。

<html>
HTMLのルート
HTMLファイル全体を包む最も外側のタグ。すべての要素はこの中に入ります。
<html lang="ja">…</html>
<head>
ヘッド（文書情報置き場）
タイトル・CSSリンク・文字コードなど、ページの設定情報をまとめる場所。
<head>…</head>
<title>
ページタイトル
ブラウザのタブに表示されるタイトルを指定します。検索結果にも使われます。
<title>私のサイト</title>
<meta>
メタ情報
文字コード・ビューポート・ページの説明など、様々な設定を1行で記述できます。
<meta charset="UTF-8"> /* 閉じタグ不要 */
<link>
外部リソース接続
主にCSSファイルをHTMLに読み込むために使います。ファビコンの指定にも使用。
<link rel="stylesheet" href="style.css">
<style>
インラインCSS
HTMLファイルの中に直接CSSを書く場合に使います。小規模な装飾に便利。
<style> body { color: black; } </style>
🏗️

ページ構造・区分

ページのレイアウトを論理的に分けるための要素です。見た目だけでなく、アクセシビリティや SEO にも影響します。

<body>
本文エリア
ブラウザに実際に表示されるコンテンツをすべて含む部分。1ページに1つだけ。
<header>
ヘッダー（上部）
ページ上部のロゴ・タイトル・ナビゲーションをまとめる場所。
<nav>
ナビゲーション
メニューリンクなど、サイト内を移動するためのリンクをまとめます。
<main>
メインコンテンツ
そのページの主役となるコンテンツを囲みます。1ページに1つ。
<section>
セクション（章・節）
ページを意味のある区切りで分けます。見出しと一緒に使うのが基本。
<article>
記事・独立したコンテンツ
ブログ記事・ニュース・コメントなど、それだけで完結するコンテンツに使います。
<aside>
サイドコンテンツ
本文と間接的に関連する内容（サイドバー・補足情報・広告など）を置きます。
<footer>
フッター（下部）
ページ下部に置く著作権・連絡先・補足リンクなどをまとめます。
<div>
汎用コンテナ（ブロック）
特に意味を持たない「箱」。CSSでスタイルを当てたいときにグループ化するために使います。
<h1>〜<h6>
見出し（レベル1〜6）
h1が最大の見出し、h6が最小。ページ内の階層構造を作ります。h1は通常1ページに1つ。
<h1>大見出し</h1> <h2>中見出し</h2> <h3>小見出し</h3>
✏️

テキストコンテンツ（ブロック要素）

段落・リスト・引用など、文章を構成するための要素です。

<p>
段落（パラグラフ）
ひとまとまりの文章を囲みます。最もよく使うテキスト要素のひとつ。
<p>こんにちは、世界！</p>
<ul>
箇条書きリスト（順序なし）
・マークが付く一般的な箇条書き。<li>と組み合わせて使います。
<ul><li>りんご</li><li>みかん</li></ul>
<ol>
番号付きリスト（順序あり）
1, 2, 3…と番号が付くリスト。手順や順位を表すときに使います。
<ol><li>まず材料を揃える</li><li>次に混ぜる</li></ol>
<li>
リストの各項目
<ul>または<ol>の中に入るリストの1項目を表します。
<blockquote>
引用文（ブロック）
他のソースから引用した長い文章を表します。インデントされて表示されます。
<pre>
整形済みテキスト
スペースや改行をそのまま表示します。コードの表示によく使われます。
<hr>
水平線（区切り線）
話題の切り替えを示す水平方向の区切り線を引きます。閉じタグは不要。
<hr> /* 閉じタグ不要 */
🔤

インラインテキスト（文中に使う要素）

文章の一部に意味や装飾を加えます。段落（p）などの中で使います。

<a>
リンク（アンカー）
クリックして他のページや場所に移動するリンクを作ります。href属性でリンク先を指定。
<a href="https://example.com">クリックしてね</a>
<strong>
強調（重要・太字）
重要な部分を太字で表示します。見た目だけでなく「重要」という意味を持ちます。
これは<strong>とても重要</strong>です。
<em>
強調（イタリック）
文章の中でアクセントを付けたい部分を斜体で表示します。
<span>
汎用コンテナ（インライン）
文章の一部にCSSを当てたいときに使う「意味を持たない」インライン要素。
文字を<span style="color:red">赤</span>にする
<code>
インラインコード
文中にコードの断片を示すときに使います。等幅フォントで表示されます。
<code>console.log()</code> を使います。
<br>
改行
文章の途中で強制的に改行します。詩や住所など、改行位置が意味を持つ場合に。
1行目<br>2行目 /* 閉じタグ不要 */
🖼️

画像・マルチメディア

画像・動画・音声をページに埋め込むための要素です。

<img>
画像
画像を表示します。src属性で画像ファイルを、alt属性で代替テキストを指定します（アクセシビリティのため必須）。
<img src="photo.jpg" alt="猫の写真">
<video>
動画
動画ファイルをページに埋め込みます。controls属性で再生ボタンを表示できます。
<video src="movie.mp4" controls></video>
<audio>
音声
音声ファイルを埋め込みます。controls属性で再生コントロールを表示します。
<figure>
図版コンテナ
画像・グラフ・コードブロックなど、キャプション付きにしたいコンテンツを包みます。
<figure><img src="…" alt="…"><figcaption>図1</figcaption></figure>
📝

フォーム

ユーザーが入力・送信するためのUI部品です。

<form>
フォーム全体
入力項目と送信ボタンをひとまとめにする外側のコンテナ。
<input>
入力フィールド
テキスト・パスワード・チェックボックス・ラジオボタンなど、type属性で種類を変えられる万能フォーム部品。
<input type="text" placeholder="名前を入力">
<textarea>
複数行テキスト入力
お問い合わせ内容など、長い文章を入力するための複数行テキストエリア。
<button>
ボタン
クリックできるボタンを作ります。フォームの送信や、JavaScriptの処理起動に使います。
<button>送信する</button>
<label>
ラベル
入力欄の名前を示します。for属性でinputと紐付けると、ラベルをクリックしたときに入力欄がフォーカスされます。
<label for="name">お名前</label><input id="name">
<select>
ドロップダウン選択
プルダウンメニュー形式で選択肢を表示します。中に<option>を並べます。
📊

表（テーブル）

データを行・列で整理して見せるための要素です。

<table>
表全体
表を作るための外側のコンテナ。この中に行・列・セルを並べます。
<tr>
行（テーブルロウ）
表の横1行を表します。この中に<th>や<td>を並べます。
<th>
見出しセル
表のヘッダーセル。太字・中央揃えで表示され、列や行の名前を示します。
<td>
データセル
表の通常のデータを入れるセル。<tr>の中に並べます。
<table><tr><th>名前</th><th>年齢</th></tr><tr><td>田中</td><td>20</td></tr></table>
