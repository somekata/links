§1　確率の基礎

確率の定義・加法定理・乗法定理・条件付き確率

Q1 確率の定義
サイコロを1個振るとき、偶数の目が出る確率として正しいのはどれか。
✓ 正解 偶数の目は 2・4・6 の3通り。全事象は 1〜6 の6通り。
P(偶数) = 3 ÷ 6 = 1/2 古典的確率の定義：P(A) = A が起こる場合の数 ÷ 全事象の数
Q2 加法定理
P(A) = 0.4、P(B) = 0.3、P(A∩B) = 0.1 のとき、P(A∪B) はいくらか。
✓ 正解 加法定理（一般形）を使う。
P(A∪B) = P(A) + P(B) − P(A∩B) = 0.4 + 0.3 − 0.1 = 0.6 AとBが排反でない場合、重複部分（A∩B）を一度引く必要がある。
Q3 条件付き確率
ある病院の患者のうち、糖尿病患者は20%。HbA1c高値（>6.5%）の患者は15%。糖尿病かつHbA1c高値の患者は12%。
糖尿病である患者の中で、HbA1c が高値である条件付き確率 P(HbA1c高|糖尿病) はいくらか。
✓ 正解 条件付き確率の定義式を使う。
P(HbA1c高|糖尿病) = P(HbA1c高 ∩ 糖尿病) ÷ P(糖尿病) = 0.12 ÷ 0.20 = 0.60 「糖尿病という条件のもとで空間を絞り込み、その中でHbA1c高値が占める割合」を計算する。
Q4 独立・排反
以下の記述のうち、正しいものはどれか。
✓ 正解 排反とは「同時に起こらない」こと（P(A∩B)=0）。
独立とは「一方の結果が他方に影響しない」こと（P(A|B)=P(A)）。

確率が0でない排反事象では、「A が起きた → B は絶対起きない」という影響関係がある。
これは独立の定義に反する。つまり排反事象は独立ではない。

§2　確率変数・期待値・分散

E[X]・Var(X)の計算と性質

Q5 期待値の計算
確率変数 X が次の分布に従うとき、E[X] はいくらか。

X0123
P0.20.40.30.1
✓ 正解 E[X] = 0×0.2 + 1×0.4 + 2×0.3 + 3×0.1 = 0 + 0.4 + 0.6 + 0.3 = 1.3 期待値は「各値 × その確率」を全て足したもの。確率の合計は 0.2+0.4+0.3+0.1=1.0 で確認できる。
Q6 期待値の性質
E[X] = 5、Var(X) = 4 のとき、Y = 3X + 2 に対して E[Y] と Var(Y) はいくらか。
✓ 正解 E[Y] = E[3X+2] = 3×E[X] + 2 = 3×5 + 2 = 17 Var(Y) = Var(3X+2) = 3²×Var(X) = 9×4 = 36 定数を足しても分散は変わらない。定数を掛けると分散は2乗倍になる点に注意。
標準偏差なら SD(Y) = 3×SD(X) = 3×2 = 6。
Q7 分散の意味
患者AとBの血糖値（mg/dL）を5回ずつ測定した。
A：110, 110, 110, 110, 110（平均110）
B：90, 100, 110, 120, 130（平均110）
2人の分散について正しい記述はどれか。
✓ 正解 A の全値は平均(110)と完全に一致 → 各偏差²=0 → Var(A)=0
B の偏差は −20, −10, 0, +10, +20 →
Var(B) = (400+100+0+100+400) ÷ 5 = 200 平均が同じでも分散（ばらつき）は全く異なる。分散はデータの「不確実性」や「変動の大きさ」を表す。

§3　検定の選択

問いの形から適切な検定を選ぶ

Q8 検定の選択
同じ患者20人に対して、治療前・治療後の疼痛スコア（0〜10の連続値、正規分布に従う）を比較したい。最も適切な検定はどれか。
✓ 正解 検定選択の3ステップで考える：
①変数型 → 連続変数（疼痛スコア）
②群数 → 2群（治療前・後）
③対応 → あり（同じ患者を繰り返し測定）

正規分布に従う連続変数で対応ありの場合 → 対応のある t 検定
「独立2標本 t 検定」は別々の患者群を比べるとき。対応があるのに独立検定を使うと検出力が下がる。
Q9 カテゴリ検定
新薬群（n=10）とプラセボ群（n=10）で、副作用発生の有無（あり/なし）を比較する。新薬群での副作用発生者は2名、プラセボ群では1名だった。最も適切な検定はどれか。
✓ 正解 カテゴリ変数（副作用あり/なし）・対応なし → χ² 検定か Fisher 正確検定。

χ² 検定の適用条件：全セルの期待度数が5以上。
この例は小標本（n=10ずつ）で副作用が少なく、期待度数が5未満のセルが生じる → Fisher 正確検定 が適切。

McNemar 検定は対応ありの2値比較（治療前後など）に使う。
Q10 パラ vs ノンパラ
パラメトリック検定とノンパラメトリック検定の比較として、正しい記述はどれか。
✓ 正解 パラメトリック検定は「正規分布」などの仮定を必要とする代わりに、仮定が正しければ検出力（真の差を見つける力）が高い。

ノンパラメトリック検定は分布の形を仮定しないため頑健で外れ値にも強いが、その分やや保守的（検出力が低め）。

「どちらが常に優れている」というわけではなく、データの特性に合わせて選ぶのが正解。

§4　生存分析・ベイズ統計

時間イベントデータとベイズ的思考

Q11 生存分析
生存分析における「打ち切り（censoring）」の説明として正しいのはどれか。
✓ 正解 打ち切りとは「観察終了時にまだイベントが起きていない」状態。例：試験終了時点で生存中の患者。

このデータは「少なくとも〇日間イベントなし」という情報を持っており、完全に捨てるべきではない。カプラン・マイヤー法はこの情報を活かして生存曲線を推定できる点が最大の強み。

④の「追跡不能」も打ち切りとして扱われるが、打ち切りの説明としては③が最も正確。
Q12 ベイズの定理
疾患Yの有病率は5%。検査Zの感度は90%、特異度は80%。ある患者の検査Zが陽性だった。この患者が疾患Yである確率（陽性的中率）に最も近いのはどれか。
✓ 正解 1000人で計算する。
疾患あり：50人、疾患なし：950人
真陽性：50 × 0.90 = 45人
偽陽性：950 × (1−0.80) = 190人
PPV = 45 ÷ (45 + 190) = 45 ÷ 235 ≈ 19% 感度・特異度が高くても有病率が低ければ PPV は低くなる。これがベイズ的思考の核心——事前確率（有病率）が事後確率（PPV）を大きく左右する。

§5　診断の統計

感度・特異度・尤度比・ROC

Q13 感度・特異度
「感度が高い検査では陰性であれば疾患を除外できる（SnNout）」の理由として正しいのはどれか。
✓ 正解 感度 = TP ÷ (TP + FN)。感度が高い = FN（偽陰性）が少ない。

つまり「本当に病気なのに陰性」という見逃しがほとんどない。
→ 陰性が出たなら「見逃しではない」→ 疾患除外の根拠になる。

SnNout：Sensitivity high → Negative rules out（除外に使う）
SpPin：Specificity high → Positive rules in（確定に使う）
Q14 ROC・AUC
ROC曲線とAUCについての記述のうち、正しいのはどれか。
✓ 正解 ROC曲線は「カットオフ値を動かしたときの感度 vs（1−特異度）の軌跡」。

AUC（Area Under the Curve）の解釈：
1.0 = 完璧な識別　0.5 = コイントスと同じ（識別力なし）
0.8以上 = 良好な検査、0.7〜0.8 = 許容範囲、0.7未満 = 識別力が低い

カットオフ値を変えると感度と特異度が変わるので、曲線上の「どこに位置するか」が変わるが、AUC（曲線の形そのもの）は変わらない。
Q15 総合問題
次の2×2表から、感度・特異度・陽性的中率を求めた。正しい組み合わせはどれか。

疾患あり疾患なし
検査陽性8020
検査陰性20180
✓ 正解 TP=80、FP=20、FN=20、TN=180
感度 = TP÷(TP+FN) = 80÷(80+20) = 80÷100 = 80% 特異度 = TN÷(TN+FP) = 180÷(180+20) = 180÷200 = 90% PPV = TP÷(TP+FP) = 80÷(80+20) = 80÷100 = 80% 感度と PPV がともに80%になっているが、計算の分母が異なる点に注意。感度の分母は「疾患あり全員」、PPV の分母は「検査陽性全員」。
