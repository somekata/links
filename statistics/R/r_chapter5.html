CHAPTER 5 ── 第5章

検定のロジック
〜「偶然」を切り捨てる勇気〜

差があるように見えるのは「本当の差」か「偶然のゆらぎ」か——仮説検定はその問いに答える道具だ。p値の本当の意味から、効果量・検定力まで掘り下げる。

📌 この章の学習目標

  • 帰無仮説・対立仮説・p値の概念を正確に説明できる
  • 第1種過誤（α）と第2種過誤（β）の違いと関係を説明できる
  • 対応なし・あり t検定を適切に使い分け、Rで実行できる
  • ウィルコクソン検定・マン・ホイットニー検定を使える
  • カイ二乗検定・フィッシャーの正確検定を実行できる
  • 一元配置・二元配置ANOVAを実行し、多重比較を行える
  • 効果量（Cohen's d）と検定力（power）を計算できる
  • p値の限界と「統計的有意性」の誤用について議論できる

5.1仮説検定のロジック ── 「偶然」を疑う

師
先生 ある新薬を使った50人のHbA1cが平均7.1%、対照群50人が平均7.4%だったとする。この差0.3%は「新薬が効いた証拠」だろうか？　それとも「たまたまそういうサンプルを引いた」という偶然だろうか？　仮説検定はこの問いに答える論理の枠組みだ。
学
学生 差が大きいほど「偶然じゃない」と言えそうですが、どう定量化するんですか？
師
先生 逆向きに考えるんだ。「もし本当は差がないとしたら（帰無仮説）、今回のような差が偶然生じる確率はどれくらいか？」——この確率がp値だ。p値が小さいほど「帰無仮説のもとではめったに起きないこと」が起きたことを示す。慣例としてp < 0.05 を「有意」とみなして帰無仮説を棄却する。
1
帰無仮説（H₀）を立てる：「新薬群と対照群のHbA1cに差はない」
2
対立仮説（H₁）を立てる：「新薬群のほうがHbA1cが低い（または差がある）」
3
検定統計量を計算する：t値・χ²値など。データの差を「標準誤差」で割った比として表現される
4
p値を求める：帰無仮説が正しいとき、この検定統計量以上に極端な値が出る確率
5
判断する：p < 0.05 なら H₀ を棄却→「統計的に有意な差がある」。p ≥ 0.05 なら H₀ を棄却できない
p値の正確な定義： 「帰無仮説が真であるという前提のもとで、観察されたデータ以上に極端なデータが生じる確率」。「差がある確率」でも「帰無仮説が正しい確率」でもない。

5.2第1種・第2種の過誤 ── 間違いの2種類

師
先生 検定は間違いを犯すことがある。その間違いには2種類ある。どちらを重視するかは、研究の文脈によって変わってくる。
第1種過誤（Type I Error）
確率 = α（有意水準）
「本当は差がないのに、差があると結論する」。フォールス・ポジティブ（偽陽性）。α = 0.05 に設定することで、この誤りを5%以下に抑える。新薬が本当は無効なのに「有効」と判定してしまう危険。
第2種過誤（Type II Error）
確率 = β（通常 0.1〜0.2）
「本当は差があるのに、差がないと結論する」。フォールス・ネガティブ（偽陰性）。検定力 = 1 - β。有効な薬を「効かない」と見逃す危険。サンプルサイズを増やすと β は減る。

⚖️ α と β のトレードオフ

α を厳しくすると（0.05→0.01）、第1種過誤は減るが第2種過誤が増える。双方を同時に下げるにはサンプルサイズを増やすしかない。このバランスをどう設定するかが「検定力計算（power calculation）」の本質であり、研究計画の段階で必ず行うべき作業だ。

R# ── サンプルサイズ計算（pwr パッケージ）──
library(pwr)

# 2群のt検定：効果量 d=0.5、α=0.05、検定力=0.8 に必要なn
pwr.t.test(
  d    = 0.5,      # Cohen's d（中程度の効果量）
  sig.level = 0.05,
  power     = 0.80,
  type      = "two.sample"
)
# n = 63.77 → 各群64人必要

# 逆に：n=50, d=0.5 のときの検定力は？
pwr.t.test(n = 50, d = 0.5, sig.level = 0.05, type = "two.sample")
# power = 0.6970 → 70%の検定力（やや不足）

5.3t検定 ── 2群の平均を比較する

師
先生 最もよく使う検定がt検定だ。「2群の平均に差があるか」を問う。対応なし（独立2群）と対応あり（paired）の2種類がある。「対応あり」は同一患者の治療前後など、個体が対になっているケースで使う。 
R# ── 対応なし t検定（independent samples）──
# 男女のHbA1cに差があるか？
t_result <- t.test(
  hba1c ~ sex,
  data    = patients,
  var.equal = FALSE   # Welchのt検定（等分散を仮定しない）
)
print(t_result)
# t = -1.84, df = 95.3, p-value = 0.0688
# 95% CI: -0.52 to 0.02
# 有意水準0.05では有意差なし

# ── 等分散の確認（Levene 検定）──
library(car)
leveneTest(hba1c ~ sex, data = patients)

# ── 対応あり t検定（paired samples）──
# 治療前後のHbA1c変化
t_paired <- t.test(
  x       = patients$hba1c_post,
  y       = patients$hba1c_pre,
  paired  = TRUE,
  alternative = "less"    # 一側検定：治療後の方が低いか
)
print(t_paired)

# ── 結果を tidyで取り出す（broom パッケージ）──
library(broom)
tidy(t_result)
# estimate statistic p.value conf.low conf.high method
#    -0.248    -1.838  0.0688   -0.518    0.021  Welch Two Sample t-test

# ── グラフと検定をセットで（ggpubr パッケージ）──
library(ggpubr)
patients |>
  ggboxplot(x = "sex", y = "hba1c",
            fill = "sex",
            palette = c("#2a6b6e", "#c8922a"),
            add = "jitter") +
  stat_compare_means(method = "t.test",
                     label = "p.format")
Welch vs Student の t検定： Rのデフォルトは var.equal = FALSE（Welch法）。等分散を仮定しないWelch法はより頑健で、等分散が成り立っている場合でも大きな損失がない。よほどの理由がなければWelch法を使うことを推奨する。

5.4ノンパラメトリック検定 ── 正規性を仮定しない

師
先生 t検定は正規分布を前提とする。しかし医学データの多くは非正規だ——入院日数・医療費・CRPなど。こういうケースではノンパラメトリック検定が安全だ。データの「値」ではなく「順位」に基づくので、分布の仮定が不要だ。 
R# ── マン・ホイットニーのU検定（対応なし2群）──
# Wilcoxon rank-sum test とも呼ばれる
wilcox.test(
  hba1c ~ sex,
  data = patients,
  exact = FALSE   # タイがある場合は exact=FALSE
)
# W = 1089, p-value = 0.0534

# ── ウィルコクソンの符号付き順位検定（対応あり）──
wilcox.test(
  x      = patients$hba1c_post,
  y      = patients$hba1c_pre,
  paired = TRUE,
  alternative = "less"
)

# ── クラスカル・ウォリス検定（3群以上のノンパラ）──
kruskal.test(hba1c ~ age_group, data = patients)
# Kruskal-Wallis chi-squared = 8.43, df = 2, p-value = 0.0148

# 多重比較（Dunn 検定）
library(dunn.test)
dunn.test(patients$hba1c, patients$age_group, method = "bonferroni")
状況パラメトリックノンパラメトリック代替
独立2群の比較 正規 t.test(var.equal=F) 順位 wilcox.test()
対応2群の比較 正規 t.test(paired=T) 順位 wilcox.test(paired=T)
3群以上の比較 正規 aov() 順位 kruskal.test()
対応3群以上 正規 aov() + Error(id) 順位 friedman.test()

5.5カイ二乗検定と比率の検定 ── カテゴリ変数を比べる

師
先生 「男女で糖尿病の合併症の有無に差があるか」など、カテゴリ変数同士の関連を調べるにはカイ二乗検定を使う。クロス集計表（分割表）を作って、期待度数と観察度数のズレを検定する。 
R# ── カイ二乗検定 ──
# インスリン使用率は性別で異なるか？
tbl <- table(patients$sex, patients$on_insulin)
print(tbl)
#      FALSE TRUE
#   F    28   17
#   M    32   23

chi_result <- chisq.test(tbl)
print(chi_result)
# X-squared = 0.228, df = 1, p-value = 0.633

# 期待度数の確認（期待度数が5未満のセルがある場合はFisherの正確検定へ）
chi_result$expected

# ── フィッシャーの正確検定（小サンプル時）──
fisher.test(tbl)

# ── 2比率のZ検定（prop.test）──
# A病院50人中12人合併症、B病院80人中28人合併症
prop.test(
  x = c(12, 28),
  n = c(50, 80),
  correct = FALSE
)
# prop 1 = 0.24, prop 2 = 0.35
# p-value = 0.182 → 有意差なし

# ── 分割表の可視化（モザイクプロット）──
patients |>
  filter(!is.na(on_insulin)) |>
  count(sex, on_insulin) |>
  ggplot(aes(x = sex, y = n, fill = on_insulin)) +
  geom_col(position = "fill") +
  scale_fill_manual(values = c("#2a6b6e", "#c8922a"),
                     labels = c("非使用", "使用")) +
  scale_y_continuous(labels = scales::percent) +
  labs(title = "性別別インスリン使用率",
       y = NULL, fill = "インスリン") +
  theme_classic()

5.6分散分析（ANOVA） ── 3群以上の平均を同時に比較

師
先生 3つの治療群を比較したいとき、t検定を3回繰り返してはいけない。比較を繰り返すたびに偶然有意になる確率が膨らむからだ（多重比較の問題）。3群以上を一度に比較するのがANOVA（分散分析）だ。 
R# ── 一元配置ANOVA ──
# 年齢3群でHbA1cに差があるか
patients <- patients |>
  mutate(age_group = case_when(
    age < 40           ~ "40未満",
    age >= 40 & age < 60 ~ "40-59",
    age >= 60           ~ "60以上"
  ))

anova_result <- aov(hba1c ~ age_group, data = patients)
summary(anova_result)
#            Df Sum Sq Mean Sq F value Pr(>F)
# age_group   2  12.45   6.23   3.41  0.037 *
# Residuals  97 177.3    1.83

# F検定が有意 → どの群間に差があるかを多重比較で調べる

# ── Tukey の HSD（多重比較）──
TukeyHSD(anova_result)
# $age_group
#              diff   lwr   upr  p adj
# 40-59-40未満 0.38 -0.18  0.94  0.256
# 60以上-40未満 0.72  0.12  1.32  0.014 *
# 60以上-40-59  0.34 -0.18  0.86  0.283

# ── 二元配置ANOVA（交互作用あり）──
anova2 <- aov(hba1c ~ sex * on_insulin, data = patients)
summary(anova2)

# ── 交互作用プロット ──
patients |>
  filter(!is.na(on_insulin)) |>
  group_by(sex, on_insulin) |>
  summarise(mean_hba1c = mean(hba1c, na.rm = TRUE), .groups = "drop") |>
  ggplot(aes(x = on_insulin, y = mean_hba1c,
             color = sex, group = sex)) +
  geom_line(linewidth = 1) +
  geom_point(size = 3) +
  scale_color_manual(values = c(M = "#2a6b6e", F = "#c8922a")) +
  labs(title = "交互作用プロット",
       x = "インスリン使用", y = "HbA1c 平均") +
  theme_classic()

5.7多重比較の問題 ── 検定を繰り返す危険

師
先生 α = 0.05 で検定を20回繰り返すと、本当は差がなくても1回は偶然「p < 0.05」になる計算だ。これが多重比較問題だ。補正には主にBonferroni法とBenjamini-Hochberg（BH）法が使われる。 
R# ── p値の多重比較補正 ──
# 例：5つの検定を行ったp値
p_values <- c(0.023, 0.041, 0.003, 0.310, 0.048)

# Bonferroni 補正（保守的、familywise error rateを制御）
p.adjust(p_values, method = "bonferroni")
# [1] 0.115 0.205 0.015 1.000 0.240

# BH 補正（FDR制御、複数仮説検定に推奨）
p.adjust(p_values, method = "BH")
# [1] 0.058 0.068 0.015 0.310 0.068

# ── emmeans による多重比較（ANOVAの後処理）──
library(emmeans)
emm <- emmeans(anova_result, ~ age_group)
pairs(emm, adjust = "tukey")

# Compact letter display（どの群が有意に異なるかを「a, b, c」で表示）
library(multcomp)
cld(emm, Letters = letters)
Bonferroni vs BH の使い分け： 1つでも誤りを犯してはいけないケース（臨床試験の主要評価項目）にはBonferroni。多くの仮説を探索的に検定する場合（ゲノムワイド解析・プロテオミクス）にはBH法（偽発見率制御）が適している。

5.8効果量と検定力 ── 「有意」だけでは足りない

師
先生 10万人の大規模研究でHbA1cの差が0.01%でもp < 0.001になる。しかしその差は臨床的に意味があるか？　p値はサンプルサイズの影響を強く受ける。「差の大きさ」を表す指標が効果量だ。 
R# ── Cohen's d（2群の平均差を標準化）──
library(effectsize)

cohens_d(hba1c ~ sex, data = patients)
# Cohen's d |       95% CI
# ----------|-------------
#     -0.37 | [-0.77, 0.03]
# d = 0.2: 小, 0.5: 中, 0.8: 大（Cohenの目安）

# ── η²（イータ二乗）：ANOVAの効果量 ──
eta_squared(anova_result)
# Parameter | Eta2 |       95% CI
# ---------|------|---------------
# age_group | 0.07 | [0.01, 0.18]
# η² ≈ 0.01: 小, 0.06: 中, 0.14: 大

# ── 検定力曲線のプロット ──
library(pwr)
n_seq <- seq(10, 150, by = 5)
power_seq <- sapply(n_seq, function(n) {
  pwr.t.test(n = n, d = 0.4, sig.level = 0.05,
             type = "two.sample")$power
})

tibble(n = n_seq, power = power_seq) |>
  ggplot(aes(x = n, y = power)) +
  geom_line(color = "#2a6b6e", linewidth = 1.2) +
  geom_hline(yintercept = 0.8, linetype = "dashed",
             color = "#c8922a") +
  annotate("text", x = 10, y = 0.82, label = "80%の検定力",
           hjust = 0, size = 3, color = "#c8922a") +
  labs(title = "検定力曲線（d = 0.4, α = 0.05）",
       x = "各群のサンプルサイズ", y = "検定力（1 - β）") +
  theme_classic()

5.9p値を超えて ── 統計的有意性の限界

師
先生 2019年、科学誌Natureに800人以上の科学者が署名した論文が発表された。主張はシンプルだ——「統計的有意性という概念の使用をやめよ」。なぜそこまで言うのか、考えてみよう。
学
学生 p < 0.05 が長年使われてきたのに、やめろというのは急進的に聞こえます。何が問題なんですか？
師
先生 問題は「p < 0.05 か否か」という二値的な判断に研究者が飛びつきすぎることだ。第一に、p = 0.049 と p = 0.051 の間に実質的な差はない。第二に、有意でも効果量が小さければ臨床的に無意味な差かもしれない。第三に、「有意でない＝差がない」ではなく「差があるとは言えない」にすぎない——これを混同している論文が多い。

📋 p値に関する6つの誤解

誤解1：「p = 0.03 は差がある確率が97%」——p値は「帰無仮説のもとで」計算される確率。「差がある確率」ではない。

誤解2：「p > 0.05 は差がない証拠」——差がないことを証明するには「等価性検定」が必要。

誤解3：「有意な差は重要な差」——効果量が小さくてもサンプルが大きければ有意になる。

誤解4：「p値が小さいほど研究の価値が高い」——再現性・効果量・バイアスのほうが重要なことが多い。

誤解5：「p = 0.049 と p = 0.051 は本質的に違う」——閾値はあくまで慣例。連続的に報告すべき。

誤解6：「検定をたくさんやれば何か有意になる」——多重比較問題。これは研究不正（p-hacking）になる。

R# ── 等価性検定（差がないことを積極的に示す）──
library(TOSTER)

# 「臨床的意味のある差 = ±0.5% 以内なら等価」とする
tsum_TOST(
  m1 = 7.4, sd1 = 1.2, n1 = 50,
  m2 = 7.3, sd2 = 1.3, n2 = 50,
  low_eqbound = -0.5,
  high_eqbound =  0.5,
  alpha = 0.05
)

# ── 推奨される報告スタイル（効果量 + CI を主役に）──
library(broom)
library(effectsize)

t_res  <- t.test(hba1c ~ sex, data = patients)
d_res  <- cohens_d(hba1c ~ sex, data = patients)

sprintf(
  "差 = %.2f (95%%CI: %.2f to %.2f), t(%d) = %.2f, p = %.3f, d = %.2f",
  diff(t_res$estimate),
  t_res$conf.int[1], t_res$conf.int[2],
  round(t_res$parameter),
  t_res$statistic,
  t_res$p.value,
  d_res$Cohens_d
)
    

      現代の推奨スタイル： p値は「0.047」のように連続値で報告する。効果量と95%CIを必ず併記する。「有意な差があった（p = 0.03）」ではなく「HbA1cは治療群で平均0.4%（95%CI: 0.1〜0.7）低下した（t(96) = 2.8, p = 0.006, Cohen's d = 0.28）」のように記述する。

5.10練習問題

🧠 問題1：検定の選択

以下の各状況に適切な検定を選び、Rで実行せよ。(1) 男女のBMIに差があるか（正規性は確認済み）。(2) インスリン使用前後のHbA1c変化（同一患者、非正規分布）。(3) 年齢3群と合併症の有無（カテゴリ×カテゴリ）。

R# (1) 対応なしt検定（正規性確認済み）
t.test(bmi ~ sex, data = patients, var.equal = FALSE)

# (2) ウィルコクソン符号付き順位検定（対応あり・非正規）
wilcox.test(patients$hba1c_post, patients$hba1c_pre,
            paired = TRUE, alternative = "less")

# (3) カイ二乗検定（カテゴリ×カテゴリ）
tbl <- table(patients$age_group, patients$complication)
chisq.test(tbl)
# 期待度数が5未満のセルがある場合はフィッシャーの正確検定へ

🧠 問題2：ANOVA + 多重比較
