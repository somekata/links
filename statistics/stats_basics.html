Ch. 1

確率とは何か

「可能性を数で表す」という発想がどこから来るのか、直感から始めよう。

学生
そもそも「確率」って、何ですか？ なんとなくわかるけど、ちゃんと説明できないんですよね。
先生
うん、正直に言えたね。じゃあ聞くけど、コインを投げて表が出る確率は？
学生
1/2…ですよね。
先生
そう。なぜそう思った？
学生
表か裏かの2通りで、どっちも同じくらい出そうだから…
先生
それが核心だよ。確率というのは、「全体のうち、注目している結果がどれくらい起こりやすいか」を 0〜1 の数で表したもの。0 は絶対起こらない、1 は必ず起こる。コインは「表」と「裏」が同じだけ起こりやすいから、各1/2 というわけ。
学生
じゃあ、サイコロで 6 が出る確率は 1/6？
先生
正解。全部で6つの面があって、「6」は1つだから 1/6。これを古典的確率という。「同様に確からしい場合の数」で割るやり方だね。
📐 定　義
P(A) = A が起こる場合の数 ÷ 全事象の数

ただし、各事象が「同様に確からしい」ことが前提。
確率 P(A) は常に 0 ≦ P(A) ≦ 1 を満たす。

⚠ ポイント
「同様に確からしい」が成り立たない場合は、古典的確率の定義が使えない。そのときは繰り返し実験や統計的推定を使う（頻度論的確率）。
Ch. 2

確率の基本ルール

加法定理と乗法定理。確率計算の2本柱。

学生
サイコロで「1 か 2 が出る確率」は 2/6 ですよね？ 1/6 と 1/6 を足した。
先生
そう、それが加法定理の基本形。「または（OR）」のときは確率を足す。ただし条件がある。
学生
条件？
先生
「1 が出ると同時に 2 が出る」ことはあるかな？
学生
ないです。1回の投げでどちらかしか出ない。
先生
これを排反という。排反なら単純に足せる。でも「偶数 か 3の倍数」だと、6 が両方に入るから、単純に足すと 6 を二重に数えてしまう。
学生
じゃあ、重なった分を引けばいい？
先生
完璧。P(A または B) = P(A) + P(B) − P(A かつ B) ——これが一般の加法定理だよ。
📐 加法定理（一般形）
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

A と B が排反のとき： P(A ∩ B) = 0 なので
P(A ∪ B) = P(A) + P(B)

学生
「かつ（AND）」のときはどう計算するんですか？
先生
コインを2回投げて、2回とも表が出る確率は？
学生
1/2 × 1/2 = 1/4？
先生
そう、それが乗法定理。2回のコイン投げは独立だから、掛け算が使える。「独立」というのは、1回目の結果が2回目に影響しない、ということだよ。
📐 乗法定理（独立な場合）
P(A ∩ B) = P(A) × P(B) ← A と B が独立のとき

独立でない場合は条件付き確率を使う（次章）。

Ch. 3

独立・排反・条件付き確率

医学でよく使う「条件付き確率」の概念を丁寧に。

学生
「独立」と「排反」ってごっちゃになりそうで…
先生
混乱しやすいよね。整理しよう。
排反：同時に起こらない。コインで「表と裏」は同時に出ない。
独立：片方の結果がもう片方に影響しない。コインを2回投げる。
学生
じゃあ、排反なのに独立ってありますか？
先生
鋭い質問。確率が 0 でない事象は、排反なら独立になれない。なぜなら「A が起きた」なら「B は絶対起きない」——それはB への影響だから。
学生
あ、排反の方が強い制約なんですね。
先生
そういうこと。では次——「条件付き確率」。ある患者が糖尿病だと分かったとき、HbA1c が高い確率は？ これがP(HbA1c高 | 糖尿病)という書き方だよ。
学生
「糖尿病という条件のもとで」という意味の縦棒ですね。
先生
正確に理解してる。条件付き確率は、全体の空間を絞り込むイメージ。全患者の中から「糖尿病の人だけ」を見ている。
📐 条件付き確率
P(A | B) = P(A ∩ B) ÷ P(B) ← P(B) > 0 のとき

「B が起きたという条件のもとで、A が起きる確率」
A と B が独立なら P(A | B) = P(A)（条件が意味をもたない）

🩺 医学の例

ある病院の患者 1000 人を調べた。

HbA1c 高HbA1c 正常計
糖尿病あり18020200
糖尿病なし50750800
計2307701000

P(HbA1c高 | 糖尿病あり) = 180/200 = 0.90
P(HbA1c高 | 糖尿病なし) = 50/800 = 0.063
→ 条件によって確率が大きく変わる。

Ch. 4

確率変数という考え方

数値を「ランダムに決まるもの」として扱う——統計の核心的な発想転換。

学生
「確率変数」って教科書に出てきて、名前がもう難しそうで…
先生
名前は難しそうだけど、やってることは単純。サイコロを振ったとき、出た目を X と書く。X はあるとき 1、あるとき 3、あるとき 6 になる。これが確率変数だよ。
学生
変数って言うから x みたいな未知数かと思ってました。
先生
違いはね、普通の変数は「決まっているけど知らない」、確率変数は「まだ決まっていない。でも確率はわかる」というもの。患者さんの血圧を測る前、それは確率変数だよ。測った後は、もう固定値。
学生
測る前と後で意味が違うんですね。
先生
そう。確率変数は「測る前の世界」を記述するためのツールなんだ。で、各値に確率をくっつけた表のことを確率分布という。
📊 確率分布の例（サイコロ1個）
X（出た目）123456
P(X = x)1/61/61/61/61/61/6

確率の合計は必ず 1 になる：Σ P(X = x) = 1

✦ まとめ

確率変数 X は「結果に数値を対応させる関数」。各値 x に対して確率 P(X = x) が定まり、それら全体を確率分布という。確率の合計は常に 1。

Ch. 5

期待値

「平均してどのくらい？」——確率変数の重心を求める。

学生
期待値って、期待される値ですよね。どう求めるんですか？
先生
どう思う？ まず例を言おうか。Aさんが問題を解いたら、10問中 5問正解した。Bさんは Aさんと同じ実力で、20問解くとしたら何問正解すると思う？
学生
えっと… Aさんの正解率が 50% だから、Bさんは 10問？
先生
そうそう、自然にやってるじゃない。「20問 × 50%」って計算したでしょ。それが期待値の考え方だよ。
学生
あ、じゃあ期待値って「確率 × 数」ってこと？
先生
大枠はそれ。もう少しちゃんと言うと——各結果に、その確率を掛けて、全部足し合わせたもの。サイコロでやってみようか。「各目 × その確率」を全部足すと？
学生
1×1/6 + 2×1/6 + 3×1/6 + 4×1/6 + 5×1/6 + 6×1/6 = 21/6 = 3.5
先生
完璧。サイコロの期待値は 3.5 だよ。3.5 の目は存在しないけど、長い目で見たときの平均がそこに落ち着く。
学生
なんか、「期待」って言葉の意味がわかった気がします。願望じゃなくて、「そうなりそう」ってことですよね。
先生
うまいこと言った。統計の「期待」は希望じゃなくて予測だね。
📐 期待値の定義
E[X] = Σ xᵢ · P(X = xᵢ)

各取りうる値 xᵢ に確率 P(X = xᵢ) を掛けて合計する。
E[X] は「平均」「加重平均」とも呼ばれる。記号は μ（ミュー）も使う。

🩺 医学の例：血圧の期待値

ある外来患者の収縮期血圧が次の分布に従うとする。

血圧 X (mmHg)110120130140150
P(X = x)0.100.300.350.150.10

E[X] = 110×0.10 + 120×0.30 + 130×0.35 + 140×0.15 + 150×0.10
　　= 11 + 36 + 45.5 + 21 + 15 = 128.5 mmHg

⚠ 期待値 ≠ 最頻値・中央値
期待値は「重みつき平均」。分布が偏っていると中央値・最頻値と一致しない。医学データでは正規分布でない場合も多いため、どの代表値を使うか意識することが大切。
Ch. 6

分散と標準偏差

平均だけでは語れない「ばらつき」を数で表す。

学生
期待値（平均）だけじゃだめなんですか？
先生
患者A：毎回血糖値が 110 mg/dL。患者B：100 と 120 を交互に繰り返す。平均は両方 110 だけど、同じ？
学生
Bさんの方が変動が大きくて、管理しにくそう。
先生
そう。それが分散で表せるものだよ。「平均からどれだけ離れているか」を数にする。ずれを X − μ とすると、そのまま足すとプラスとマイナスが打ち消し合うから、2乗してから平均をとる。
学生
2乗するのはキャンセルを防ぐためか。
先生
正解。患者Aなら毎回 110 − 110 = 0 だからばらつきゼロ。患者Bは (100−110)² = 100 と (120−110)² = 100 の平均で、分散 = 100。
学生
でも単位が「(mg/dL)²」になっちゃいますね。
先生
鋭い。だから分散の平方根をとって標準偏差にする。単位が元に戻って「mg/dL」になり、解釈しやすくなる。√100 = 10 mg/dL ということは、「平均から平均的に 10 mg/dL ほどブレる」とイメージできる。
📐 分散と標準偏差の定義
Var(X) = E[(X − μ)²] = Σ (xᵢ − μ)² · P(X = xᵢ)
SD(X) = σ = √Var(X)

μ = E[X]（期待値）
Var(X) の単位は「元の単位の2乗」、σ は元の単位と同じ。
記号は Var, σ²（バリアンス）/ SD, σ（シグマ）。

📊 計算例：先ほどの血圧データ（E[X] = 128.5）
xx − μ(x − μ)²P(X=x)(x−μ)²·P
110−18.5342.250.1034.225
120−8.572.250.3021.675
130+1.52.250.350.788
140+11.5132.250.1519.838
150+21.5462.250.1046.225

Var(X) = 122.75 (mmHg)²　　σ ≈ 11.1 mmHg

Ch. 7

期待値・分散の性質

計算を楽にする便利な公式たち。

学生
期待値って、定数 a を掛けたらどうなりますか？
先生
身長を cm から mm に換算するイメージ。全部の値を10倍したら平均も10倍になる。だから E[aX] = a·E[X]。
学生
定数 b を足したら？
先生
全員に 5 点加点したら平均も 5 上がる。E[X + b] = E[X] + b。これを組み合わせると E[aX + b] = aE[X] + b だね。
学生
分散は？
先生
定数を足してもばらつきは変わらない——全員に同じ点を加えても格差は変わらない。だから Var(X + b) = Var(X)。でも a 倍すると2乗が効いて Var(aX) = a²·Var(X)。
学生
だから標準偏差は SD(aX) = |a|·SD(X) ですね。2乗して平方根だから。
先生
完璧な理解。それともう一つ——計算しやすい公式がある。
📐 便利な公式一覧
E[aX + b] = a·E[X] + b
Var(aX + b) = a²·Var(X)
Var(X) = E[X²] − (E[X])² ← 計算が楽になる変形

最後の式は「2乗の期待値」から「期待値の2乗」を引いたもの。
手計算では (X−μ)² を展開してこの形を導くと楽になる。

✦ Ch.5〜7 まとめ

期待値 E[X]：確率で重みをつけた平均。長期的な「真ん中」を表す。
分散 Var(X)：平均からのズレの2乗の平均。ばらつきの大きさ。
標準偏差 σ：分散の平方根。元の単位で解釈できる。

Ch. 8

医学への架け橋

ここまでの概念が、医学統計のどこにつながるか。

学生
確率・期待値・分散を学んだけど、これって実際の医学研究でどう使うんですか？
先生
全部つながってるよ。例えば——臨床試験で「新薬を投与した患者が回復する確率 = 0.7」とわかったとする。100人に投与したら、期待回復人数は？
学生
70人。でも「ちょうど70人」になるわけじゃないですよね。
先生
そう、そのばらつきを表すのが分散だよ。このような「回復したか否か」の確率変数はベルヌーイ試行と呼ばれて、n回繰り返したときの合計は二項分布に従う。二項分布の期待値が np、分散が np(1−p)——これが次のステップだ。
学生
じゃあ今学んだことが、t検定とか正規分布とか、もっと先につながるんですね。
先生
まさに。正規分布も「期待値=平均、分散=σ²」のパラメータで完全に決まる。検定も回帰も生存分析も、全部この土台の上に立っている。今日習ったことは、見えない基礎杭なんだ。
学生
難しいと思ってたけど、つながってるとわかると面白いですね。
先生
統計はパズルみたいなもの。一つひとつのピースは単純。全体像が見えたとき、急に楽しくなるよ。
✦ 基礎編のまとめ ── この先への道

確率 → 確率変数 → 期待値・分散 という流れを学んだ。
