01R Markdown とは

R Markdown（.Rmd）は、文章・コード・実行結果を一つのファイルに統合できる文書フォーマットです。knitr がコードを実行し、Pandoc が HTML・PDF・Word などに変換します。

処理の流れ
.Rmd → knitr（コード実行）→ .md → Pandoc → HTML / PDF / docx

代表的な用途

  • 解析レポートの自動生成（データ更新 → 再Knit で即座に更新）
  • 論文・学位論文の執筆（bookdown）
  • スライド作成（xaringan, revealjs）
  • 教材・テキスト作成

02YAML フロントマター（--- ブロック）

ファイル冒頭の --- と --- で囲まれた部分が YAML フロントマターです。文書のメタ情報や出力形式を指定します。

---
title: "解析レポート"
author: "金子幸弘"
date: "`r Sys.Date()`"
output:
  html_document:
    toc: true               # 目次を表示
    toc_float: true        # 目次を固定
    toc_depth: 3           # 目次の深さ
    number_sections: true
    theme: "flatly"        # Bootstrap テーマ
    code_folding: "hide"   # コードを折りたたむ
    df_print: "paged"     # データフレームをページング
---

主な output オプション

キー値の例説明
toctrue / false目次の表示
toc_floattrue / false目次を画面左に固定
themeflatly, cerulean, journal…Bootstrap テーマ
code_foldinghide / show / noneコードの折りたたみ
df_printpaged / kable / tibbleデータフレームの表示方式
fig_width7図の幅（インチ）デフォルト
fig_height5図の高さ（インチ）デフォルト
⚠ YAML のインデント
YAML はインデント（スペース）に厳格です。タブは使えません。必ずスペース 2 つでインデントしてください。

03Markdown 記法

見出し

# 見出し 1（h1）
## 見出し 2（h2）
### 見出し 3（h3）

テキスト装飾

**太字**   *イタリック*   ~~打ち消し~~
`インラインコード`
[リンクテキスト](https://example.com)

箇条書き・番号リスト

# 箇条書き（- または *）
- 項目 A
- 項目 B
  - ネスト（スペース 2 つ）

# 番号付き
1. 手順 1
2. 手順 2

表

| 変数 | 型      | 説明 |
|------|---------|------|
| age  | numeric | 年齢 |
| sex  | factor  | 性別 |

段落と改行

改行のルール
Markdown では、単純な改行は無視されます。新しい段落を作るには空行を 1 行入れてください。行末に半角スペース 2 つを入れると段落内改行になります。

04コードチャンク

バックティック 3 つ（```）で囲み、{r} と言語を指定します。

基本構造

```{r チャンク名, echo=TRUE, eval=TRUE}
# R コードをここに書く
summary(iris)
```

主要チャンクオプション

オプションデフォルト説明
echoTRUEコードを文書に表示する
evalTRUEコードを実行する
includeTRUEFALSE でコードも結果も非表示（実行はされる）
messageTRUEメッセージを表示する
warningTRUE警告を表示する
errorFALSEエラーが出ても処理を続ける
cacheFALSE結果をキャッシュする（重い処理に）
fig.width7図の幅（インチ）
fig.height5図の高さ（インチ）
dpi72解像度（印刷用は 300）
out.width—表示幅（"80%", "400px" など）

セットアップチャンク（先頭に必ず置く）

```{r setup, include=FALSE}
knitr::opts_chunk$set(
  echo    = TRUE,
  message = FALSE,
  warning = FALSE,
  fig.width  = 7,
  fig.height = 5,
  dpi = 150
)
```

インラインコード

文中に計算結果を埋め込めます。

サンプルサイズは `r nrow(df)` 名でした。
平均年齢は `r round(mean(df$age), 1)` 歳（SD = `r round(sd(df$age), 1)`）でした。

チャンク挿入ショートカット

RStudio では Ctrl+Alt+I（Mac: Cmd+Option+I）で即座に挿入できます。

05パッケージ管理

インストール

# 単体インストール

# 複数まとめて
install.packages(c("ggplot2", "dplyr", "readr", "meta"))

# アップデート

⚠ Rmd チャンク内に install.packages() を書かない
Knit するたびに実行されてしまいます。インストールは Console で行い、Rmd 内には library() だけ書くのが原則です。

インストール済み確認

# インストール済みか確認してからロード
if (!requireNamespace("ggplot2", quietly = TRUE)) {
  install.packages("ggplot2")
}

# インストール済みパッケージ一覧
installed.packages()[, "Package"]

08tibble

tibble は data.frame を改良したデータ構造で、tidyverse の中核です。

data.frame との主な違い

data.frametibble
表示全行表示（多すぎる）先頭 10 行＋型情報
列の部分一致OK（危険）NG（厳格）
1 列選択の型vector になるtibble を保つ
行名あり原則なし
library(tibble)

# 直接作成
df <- tibble(
  name  = c("Alice", "Bob", "Carol"),
  age   = c(25, 32, 28),
  score = c(88.5, 72.0, 95.3)
)

# data.frame から変換
df <- as_tibble(iris)

# 構造を確認
glimpse(df)
# Rows: 3
# Columns: 3
# $ name  <chr> "Alice", "Bob", "Carol"
# $ age   <dbl> 25, 32, 28

09dplyr 基本動詞

パイプ演算子

演算子由来備考
|>base R（R 4.1+）追加パッケージ不要。推奨
%>%magrittrtidyverse 慣習。古いコードに多い

filter() / select() / mutate()

library(dplyr)

df |>
  filter(age >= 18, sex == "female") |>   # 行を絞る
  select(name, age, score) |>              # 列を選ぶ
  mutate(                                   # 新列を作る・変換
    age_group = if_else(age < 30, "若年", "中年"),
    score_z   = (score - mean(score)) / sd(score)
  )

arrange() / summarise() / group_by()

df |>
  arrange(desc(score)) |>    # 降順ソート
  group_by(sex) |>            # グループ化
  summarise(
    n    = n(),
    mean = mean(score, na.rm = TRUE),
    sd   = sd(score,   na.rm = TRUE)
  ) |>
  ungroup()    # グループ解除を忘れずに

10ggplot2 基本構造

文法の骨格

ggplot(data, aes(x, y)) + geom_*() + theme_*() + labs()
データ → 美的マッピング → 幾何オブジェクト → テーマ → ラベルの順に積み重ねる

散布図の例

library(ggplot2)

ggplot(df, aes(x = age, y = score, color = sex)) +
  geom_point(size = 2, alpha = 0.7) +
  geom_smooth(method = "lm", se = TRUE) +
  labs(
    title = "年齢とスコアの関係",
    x = "年齢（歳）",
    y = "スコア",
    color = "性別"
  ) +
  theme_bw() +
  theme(text = element_text(family = "HiraKakuProN-W3"))  # 日本語フォント

よく使う geom

geom用途主な引数
geom_point()散布図size, alpha, shape
geom_line()折れ線グラフlinewidth, linetype
geom_bar()棒グラフ（度数）stat="count"
geom_col()棒グラフ（値）fill, width
geom_histogram()ヒストグラムbins, binwidth
geom_boxplot()箱ひげ図fill, outlier.shape
geom_violin()バイオリン図draw_quantiles
geom_smooth()回帰・平滑化曲線method, se
geom_text()テキストラベルaes(label=)
facet_wrap()層別プロット~ var, ncol

aes() の中と外

# aes() の中 → 変数でマッピング（凡例が出る）
geom_point(aes(color = sex))     # sex 列の値で色分け

# aes() の外 → 固定値（全点に同じ色）
geom_point(color = "steelblue")  # 全部 steelblue

11実践パターン集

CSV 読み込み → 整形 → ggplot の一気通貫

library(tidyverse)

read_csv("data/result.csv", locale = locale(encoding = "UTF-8")) |>
  filter(!is.na(score)) |>
  mutate(group = factor(group, levels = c("A", "B", "C"))) |>
  ggplot(aes(x = group, y = score, fill = group)) +
    geom_boxplot(alpha = 0.7) +
    labs(title = "グループ別スコア分布") +
    theme_classic() +
    theme(legend.position = "none")

表の出力（knitr::kable）

df |>
  group_by(group) |>
  summarise(n = n(), mean = round(mean(score), 1)) |>
  knitr::kable(caption = "グループ別集計", align = "lrr")

トラブルシューティング

症状原因・対処
Knit が通らないが Console は動く名前空間の古いキャッシュ → RStudio を完全再起動
日本語が文字化けするfileEncoding / locale 指定、またはファイルを UTF-8 で保存し直す
図が表示されないeval=TRUE, fig.width/height を確認。dev="ragg_png" も試す
パッケージが見つからないinstall.packages() は Console で実行。Rmd に書かない
キャッシュが古いcache=FALSE にするか、_cache フォルダを削除
