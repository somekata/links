第4回

ESBL

名前は一つ、中身は広がり続ける——「どこまでがESBLか」という問い
← 細菌楽入門トップへ

第3回でAmpC β-ラクタマーゼを扱いました。第4回はその対になる存在、ESBL（基質特異性拡張型β-ラクタマーゼ）です。ESBLは細菌楽入門・第5回でも触れましたが、今回はその名称の成り立ち、現在主流のCTX-M型がなぜ主役になったのか、そして「ESBLという概念はどこまで広げてよいのか」という、実はまだ決着のついていない議論まで踏み込みます。

あらかじめ範囲を明確にしておきます。本エピソードで主に扱うESBLは、CTX-M型を含むAmbler class Aのうち、セファロスポリン系薬（第4世代まで）を分解しうるβ-ラクタマーゼを原則とします。Class D（OXA型）やClass C由来の拡張型（ESAC）にも触れますが、これらは「概念がどこまで広がるか」を理解するための参照点として扱い、深掘りは今後の回に譲ります。文中の肩付き番号は、末尾の「引用文献」に対応しています。

01

ESBLの古典的定義

ESBL（Extended-Spectrum β-Lactamase）は、Bush-Jacoby機能分類のGroup 2be、Ambler分子分類のClass Aに位置づけられるβ-ラクタマーゼです3,13,15。基質スペクトルは、拡張スペクトルセファロスポリン（第3世代）とモノバクタム（アズトレオナム）に及びますが、セファマイシン（セフォキシチン）とカルバペネムは加水分解しないという枠に収まります2。

阻害薬に対しては、クラブラン酸・タゾバクタム・スルバクタムに感受性を示します。これは前回扱ったAmpC（これらの阻害薬に耐性）との最大の違いであり、治療選択（β-ラクタマーゼ阻害薬配合剤が効くかどうか）に直結する一点です1。

特徴ESBL（Class A）AmpC（Class C）
Bush-Jacoby分類Group 2beGroup 1
クラブラン酸・タゾバクタム感受性感受性耐性
セフォキシチン感受性感受性耐性
カルバペネム感受性両者とも通常感受性（カルバペネマーゼ非産生時）
主な遺伝的局在プラスミド染色体（誘導型）またはプラスミド
誘導性なしあり（染色体性の場合）

主な産生菌はE. coli、Klebsiella pneumoniae、K. oxytoca、Proteus mirabilisです1。

02

名称と概念の変遷——狭域から拡張へ

「ESBL」という名前自体、一直線に生まれたわけではありません。抗菌薬開発との軍拡競争の中で、呼び名が段階的に書き換えられてきた歴史があります4,5,6,7,19。

時期用語基質代表例
1940年代〜1960年代Narrow-spectrum β-lactamase（ペニシリナーゼ）ペニシリンのみ黄色ブドウ球菌のペニシリナーゼ
1960年代〜1980年代初頭Broad-spectrum β-lactamaseペニシリン系＋第1世代セファロスポリンTEM-1（1965年）、SHV-1
1980年代初頭Extended broad-spectrum β-lactamase＋第3世代セファロスポリン、アズトレオナムSHV-2（1983年）、TEM-3（1984〜87年）
1980年代中期〜現在Extended-spectrum β-lactamase（ESBL）同上（用語を簡略化）TEM型・SHV型変異株、後にCTX-M型

「Extended broad-spectrum」という当初の名称は、親酵素（broad-spectrum β-lactamase）との関係性を強調したものでしたが、冗長であったため「Extended-spectrum β-lactamase」に簡略化され、国際的に定着しました4。最初のTEM型β-ラクタマーゼであるTEM-1は、1965年にギリシャの患者テモネイラ（Temoneira）から分離された大腸菌に由来し、その名がそのまま酵素名になっています2。

03

CTX-M型の台頭——現在主流の理由

TEM型・SHV型ESBLが親酵素からの点変異で生まれたのに対し、CTX-M型はまったく異なる由来を持ちます。Kluyvera属という環境細菌から、水平遺伝子伝播によってde novoで獲得された酵素です2。名称の由来は、セフォタキシム（cefotaxime）への活性と、最初の分離地ミュンヘン（Munich）から来ています。

2000年以降、CTX-M型は世界的に急増し、市中感染の大腸菌と院内感染の肺炎桿菌の両方でTEM型・SHV型を数のうえで置き換えました9,10。現在、米国を含む世界的にCTX-M-15が最も頻度の高いESBLとされ、多くは国際的な拡散クローンであるE. coli ST131（特にClade C2）と関連しています1,11。CTX-M-14はCTX-M-15に次いで頻度が高く東南アジアで多く、CTX-M-27は日本とヨーロッパで新興し、ST131 Clade C1-M27と関連することが知られています2。

CTX-M優位の背景

  • プラスミド（IncF、IncI、IncKなど）や挿入配列（ISEcp1、orf477など）による水平遺伝子伝播
  • E. coli ST131をはじめとする国際的拡散クローンの存在（ST405、ST38、ST648、ST410、ST1193など）
  • 食用動物における抗菌薬使用や環境リザーバーの拡大
  • 国際旅行と衛生設備アクセスの格差

2000年以前、市中でのESBL産生菌の糞便保菌はほぼ知られていませんでしたが、2000年以降は世界的に有意に増加し、特に発展途上国で顕著です9。

04

セフェピムとESBL——「感受性でも使わない」という原則

⚠ IDSA 2024ガイダンスの明確な立場

IDSAは、単純性膀胱炎を除き、ESBL産生Enterobacterales（ESBL-E）感染症に対してセフェピムの使用を推奨していません。これは、in vitroの薬剤感受性試験でセフェピム感受性と判定された場合でも同じです1,12,18。

なぜ「感受性」が信用できないのか

  • ESBLはセフェピムを加水分解する能力を持つ。特にCTX-M型は標準的な接種菌量でもセフェピム耐性を示すことが多い1,8
  • TEM型・SHV型は見かけ上セフェピム感受性を示すことがあるが、接種菌量効果（inoculum effect）が顕著で、菌量を増やすと感受性が低下する16
  • 2010年以降、多くの検査室でESBL確認試験が廃止され、MICに基づく報告に移行した結果、以前なら「全セファロスポリン耐性」と報告されていた株の一部が「セフェピム感受性」として再分類されるようになった17
  • 臨床試験では、セフェピムMICが1〜2 µg/mLの株でも治療失敗が報告されている（腎盂腎炎・cUTI試験でセフェピム群が早期中止、院内肺炎試験でセフェピム群31%・イミペネム/シラスタチン群0%が臨床的失敗）1
特徴ESBL（Class A）AmpC（Class C、第3回参照）
セフェピム加水分解あり（特にCTX-M型）あり（誘導型・過剰産生型）
セフェピムin vitro感受性変動あり（TEM/SHV型で見かけ上感受性、CTX-M型で耐性が多い）通常感受性（誘導前）
IDSA推奨単純性膀胱炎を除き使用を推奨しない第一選択または代替選択肢

同じ「セフェピム」という薬剤が、ESBLでは避けるべき、AmpCでは推奨される——この対照は、ESBLとAmpCを正確に鑑別することの臨床的な重みを物語っています。腎盂腎炎・複雑性尿路感染症ではセフェピムを避け、菌血症・肺炎などの重症感染症でも感受性が示されていても推奨されません。単純性膀胱炎に限り、経験的治療としてセフェピムが開始され臨床的改善があれば変更不要とされています1。

05

どこまでがESBLか——概念拡張の議論

ESBLの多くはClass Aですが、実はClass D（OXA型）にもESBL様の性質を持つ変異株が存在し、Class C（AmpC由来）にも基質スペクトルが拡張した酵素が存在します。「ESBL」という言葉をどこまで広げてよいのか、これは決着のついていない論点です。

Class D——OXA型ESBL

OXA型β-ラクタマーゼの一部の変異株は、クラブラン酸への感受性が可変的でありながら、拡張スペクトルセファロスポリンを加水分解する性質を獲得しています。これらはClass Aの古典的ESBLとは阻害薬プロファイルが異なるため、「ESBL」と呼ぶこと自体に注意が必要です2。

Class C由来——拡張スペクトルAmpC（ESAC）

第3回で扱った通り、AmpC（Class C）の一部にもΩ-ループやH-10/R2-ループの変異によって基質スペクトルが拡張し、セフェピムや場合によりカルバペネムまで加水分解するようになった酵素があります（CMY-37、Acinetobacter baumanniiの自然AmpCなど）。PhilipponらはこれをESAC（Extended-Spectrum AmpC）と呼び、Bush-Jacoby Group 1eに位置づけることで、古典的ESBL（Class A）との混同を避けることを提案しています14。

ESACはクラブラン酸に耐性、セファマイシンにも耐性という点で、古典的ESBLとは阻害薬プロファイルが正反対です。「基質が拡張した」という現象は共通していても、酵素としての性質はまったく異なります。

拡張分類の一例——ESBL M・ESBL CARBAという提案

一部の研究者は、ESBLをESBL A（Class A）・ESBL M（Miscellaneous：Class C由来のプラスミド性AmpCやClass D）・ESBL CARBA（カルバペネムを分解する酵素）という3グループに再編する分類を提案しています2。

この提案は分子的多様性を反映する試みですが、クラブラン酸に耐性のAmpCを「ESBL」に含めたり、カルバペネム分解酵素（カルバペネマーゼ）まで「ESBL」の枠に入れたりする点で、古典的定義と衝突します。「ESBL」と聞いてクラブラン酸配合剤が効くと誤解されるリスクや、カルバペネマーゼとの境界が曖昧になるリスクが指摘されています。

06

ESBLという名称、今後どう使うべきか——結論の出ていない議論

「ESBLという言葉をどこまで使い続けるべきか」は、専門家の間でもまだ広く決着した議論ではありません。おおよそ次のような立場が並立しています。

狭義の定義を維持する立場
  • ESBL＝Ambler Class Aに限定し、Class C由来はESAC、カルバペネマーゼは別枠として明確に呼び分ける
  • IDSA・多くの主要ガイドラインが採用する実務上の姿勢1
  • 治療選択（β-ラクタマーゼ阻害薬配合剤が効くか）との対応が一目でわかる
広義・機能的定義を維持する立場
  • Livermore（2008）は、ESBLという用語自体は現代的な広い用法（TEM/SHVと似た表現型を示すがルーツの異なる酵素も含む）を維持しつつ、常に酵素ファミリーを明記する（「CTX-M型ESBL」「OXA型ESBL」のように）ことを提案4
  • 分子的多様性の広がりを一つの言葉で受け止められる
  • ただし「ESBL」を単独で使うと誤解を招くため、常に修飾語とセットで使う運用上の負荷がある

いずれの立場も一長一短があり、「厳密な定義に絞り込むか」「ESACのように名称そのものを分けるか」「広義のまま使うが修飾語を必須にするか」という3方向の解決策のうち、どれか一つに収束したとは言えないのが現状です。細菌楽としては、当面は臨床的な文脈ではClass A（クラブラン酸感受性）に限定して「ESBL」を使い、Class C由来の拡張型には「ESAC」、カルバペネム分解酵素には「カルバペネマーゼ」という言葉を使い分けるという、IDSAに近い狭義の立場を基本方針としつつ、この呼称の議論自体がまだ動いている、という前提を持っておくのがよいでしょう。

07

本エピソードで扱うESBLの範囲（再確認）

スコープの明示

05・06章で見た通り「ESBL」という言葉が指しうる範囲は文献によって揺れます。本シリーズでは今後、原則としてCTX-M型を含むAmbler class Aのうち、セファロスポリン系薬（第1〜4世代）を分解しうるβ-ラクタマーゼを「ESBL」として扱います。Class D（OXA型ESBL）やClass C由来の拡張型（ESAC）は、原則として別の回で個別に深掘りする対象とし、本回では概念整理のための参照点にとどめています。

Q

おさらいクイズ

Q1. ESBLとAmpCを鑑別する最大のポイントはどれでしょう？　①カルバペネムへの感受性　②クラブラン酸・タゾバクタムへの感受性　③グラム染色性状　④コロニーの匂い

答えを見る

②。ESBLはクラブラン酸・タゾバクタムに感受性ですが、AmpCはこれらに耐性を示します。カルバペネムへの感受性は両者とも通常保たれているため鑑別点にはなりません。

Q2. CTX-M型ESBLがTEM型・SHV型と最も異なる点はどれでしょう？

答えを見る

起源です。TEM型・SHV型は親酵素（TEM-1、SHV-1）からの点変異で生じますが、CTX-M型はKluyvera属という別の細菌から水平遺伝子伝播によりde novoで獲得された酵素です。

Q3. ESBL産生菌による菌血症で、in vitro試験でセフェピム感受性と判定された場合、IDSA 2024ガイダンスの立場として正しいのはどれでしょう？

答えを見る

単純性膀胱炎を除き、感受性が示されていてもセフェピムの使用は推奨されません。ESBLはセフェピムを加水分解しうる上、接種菌量効果やMIC測定の不正確性があり、臨床試験でも治療失敗が報告されているためです。

Q4. 「ESAC（Extended-Spectrum AmpC）」という用語が提案されている理由として最も適切なのはどれでしょう？

答えを見る

基質スペクトルが拡張したClass C由来の酵素を、古典的ESBL（Class A、クラブラン酸感受性）と混同しないようにするためです。ESACはクラブラン酸に耐性という点で、古典的ESBLとは阻害薬プロファイルが逆になります。
