薬剤耐性菌・定義と届出

問題12026年制度改正を含む

感染症法における薬剤耐性菌感染症の届出に関する記述として、正しいものを1つ選びなさい。

  1. a．メチシリン耐性黄色ブドウ球菌（MRSA）感染症は、5類感染症の全数把握対象であり、すべての医師に届出義務がある。
  2. b．バンコマイシン耐性腸球菌（VRE）感染症は、5類感染症の全数把握対象であり、診断したすべての医師に届出義務がある。
  3. c．薬剤耐性菌の保菌者（感染症を発症していない者）も、感染症法に基づく届出の対象となる。
  4. d．多剤耐性緑膿菌（MDRP）感染症は、令和8年4月の基準改正後も、引き続き基幹定点把握（届出義務のある医療機関が限定される方式）の対象である。
  5. e．ペニシリン耐性肺炎球菌（PRSP）感染症は、全数把握対象であり、すべての医療機関に届出義務がある。
解答・解説を見る
正解：b
  • a．誤り。MRSA感染症は5類感染症のうち基幹定点把握の対象で、届出義務があるのは内科・外科を有する300床以上の指定届出医療機関のみです。全数把握ではありません。
  • b．正しい。VRE感染症は、VRSA感染症・CRE感染症・薬剤耐性アシネトバクター感染症・多剤耐性緑膿菌（MDRP）感染症とともに、5類感染症の全数把握対象であり、診断したすべての医師に届出義務があります。
  • c．誤り。感染症法に基づく届出の対象は「感染症を発症した者」であり、症状のない保菌のみの状態は届出の対象外です（ただしCREなどでは院内の保菌者発見時にアウトブレイクに準じた感染対策の実施が求められます）。
  • d．誤り。多剤耐性緑膿菌（MDRP）感染症は、令和8年4月6日の基準改正により、従来の基幹定点把握から全数把握へと変更されました。あわせて名称も「薬剤耐性緑膿菌感染症」から「多剤耐性緑膿菌感染症」に変更されています。
  • e．誤り。PRSP感染症はMRSA・多剤耐性緑膿菌（改正前）・薬剤耐性アシネトバクター感染症などと並ぶ基幹定点把握の対象で、全数把握ではありません。
整理のポイント：2026年7月現在、5類の薬剤耐性菌6疾病のうち全数把握はVRSA・VRE・CRE・薬剤耐性アシネトバクター・多剤耐性緑膿菌（MDRP）の5つ、基幹定点把握はMRSA・PRSPの2つです。MDRPは令和8年4月の改正で定点→全数に移行した最新論点なので、要注意です。
  1. 厚生労働省健康・生活衛生局感染症対策部：医師及び指定届出機関の管理者が都道府県知事に届け出る基準（令和8年4月6日適用改正）
  2. 日本臨床微生物学会：現在注意すべき耐性菌の候補と概要
結核・接触者健診

問題2

結核の接触者健康診断に関する記述として、正しいものを1つ選びなさい。

  1. a．接触者健診の対象範囲は、初発患者との接触の程度にかかわらず、同一施設に所属する職員全員を一律に対象とする。
  2. b．インターフェロンγ遊離試験（IGRA）は、結核菌への曝露から陽性化するまでに個人差があり、一般に2〜3か月程度の「ウィンドウ期」があるとされる。
  3. c．初発患者との最終接触直後の健診でIGRAが陰性であれば、それ以降のフォローアップ検査は不要である。
  4. d．接触者健診における「同心円方式」とは、患者の病室から地理的に近い部屋の入院患者から順に検査対象とする方式をいう。
  5. e．ツベルクリン反応検査はBCG接種歴の影響を受けないため、IGRAより特異度が高い検査法である。
解答・解説を見る
正解：b
  • a．誤り。接触者健診は、感染性期間中の患者との接触の濃厚さ・時間・距離を考慮し、感染リスクの高い接触者から段階的に対象を広げる「同心円方式」で範囲を決定します。所属が同じというだけで一律に対象とするものではありません。
  • b．正しい。IGRAは結核菌への曝露後、免疫応答が成立して検査結果に反映されるまでに時間を要し、このウィンドウ期は一般に2〜3か月程度とされています。そのため登録直後の健診だけでは感染を否定しきれません。
  • c．誤り。ウィンドウ期があるため、登録直後の健診でIGRAが陰性でも、通常は最終接触から2〜3か月後を目安にフォローアップの再検査が計画されます（接触者集団の感染率が高い場合はさらに6か月後の再検査が推奨されることもあります）。
  • d．誤り。「同心円方式」は物理的な部屋の距離ではなく、患者との接触の濃厚さ・累積時間を基準に、最も濃厚な接触者（第一同心円）から段階的に対象を広げていく考え方です。
  • e．誤り。ツベルクリン反応検査はBCG接種歴や非結核性抗酸菌感染の影響で偽陽性が生じやすく、特異度はIGRAより低いとされています。IGRAはBCGの影響を受けにくい点が利点です。
整理のポイント：「同心円方式＝接触の濃厚さで対象を広げる」「IGRAのウィンドウ期＝2〜3か月」の2点は、接触者健診の設問で繰り返し問われる基本事項です。

微生物と疾患の組み合わせ

問題3

微生物とそれが原因となる代表的な疾患・病態の組み合わせとして、正しいものを1つ選びなさい。

  1. a．緑膿菌（Pseudomonas aeruginosa）─ 偽膜性大腸炎
  2. b．ディフィシル菌（Clostridioides difficile）─ 人工呼吸器関連肺炎
  3. c．マイコプラズマ（Mycoplasma pneumoniae）─ 非定型肺炎
  4. d．インフルエンザ菌（Haemophilus influenzae）─ 性感染症
  5. e．レジオネラ（Legionella pneumophila）─ 尿路カテーテル関連感染症
解答・解説を見る
正解：c
  • a．誤り。偽膜性大腸炎の主因はClostridioides difficileです。緑膿菌はブドウ糖非発酵性グラム陰性桿菌で、湿潤環境を好み、人工呼吸器関連肺炎やカテーテル関連血流感染など、異物・湿潤環境に関連した感染で問題となります。
  • b．誤り。人工呼吸器関連肺炎の代表的な起因菌は緑膿菌・アシネトバクター・腸内細菌目細菌などです。C. difficileは芽胞形成菌で、抗菌薬関連下痢症・偽膜性大腸炎の原因として重要です。
  • c．正しい。マイコプラズマは細胞壁を持たない非定型病原体で、若年者を中心に非定型肺炎（マイコプラズマ肺炎）の代表的な起因菌です。細胞壁がないため、βラクタム系抗菌薬は無効です。
  • d．誤り。性感染症としての細菌性尿道炎・子宮頸管炎の代表的起因菌は淋菌（Neisseria gonorrhoeae）です。インフルエンザ菌は侵襲性感染症（髄膜炎・喉頭蓋炎等）や気道感染で問題となる菌です。
  • e．誤り。尿路カテーテル関連感染症の主な起因菌は大腸菌をはじめとする腸内細菌目細菌です。レジオネラは冷却塔水や給湯設備などの水環境から吸入曝露することで、レジオネラ肺炎（在郷軍人病）を引き起こします。
整理のポイント：似た文脈で語られやすい菌ほど、原因と結果を入れ替えた誤答が作られやすい分野です。「どの環境・どの部位で増えるか」を軸に整理すると混同を防げます。
感染症の検査と治療（総合）

問題4

70歳男性。発熱を主訴に入院し、抗菌薬投与前に2セットの血液培養を採取した。結果、1セットのみからコアグラーゼ陰性ブドウ球菌（CNS）が検出され、もう1セットは陰性であった。この結果の解釈として最も適切なものを1つ選びなさい。

  1. a．2セット中1セットが陽性であるため、直ちに真の菌血症と判定し、広域抗菌薬による治療を開始する。
  2. b．CNSは皮膚の常在菌であり、2セット中1セットのみの検出は、血管内カテーテルの有無や敗血症症状の有無等の臨床所見と合わせて、皮膚常在菌によるコンタミネーション（検体汚染）の可能性を検討する必要がある。
  3. c．CNSは病原性を持たない菌であるため、検出されても臨床的な意味はなく、無視してよい。
  4. d．血液培養は1セットの採取で十分な感度が得られるため、複数セットの採取は不要である。
  5. e．片方のセットが陰性であった時点で、陽性となったセットの結果自体に検査上の誤りがあると判断してよい。
解答・解説を見る
正解：b
  • a．誤り。CNSは血液培養で最も頻度の高いコンタミネーション起因菌であり、2セット中1セットのみの検出という結果だけで真の菌血症と即断するのは適切ではありません。
  • b．正しい。CNSは皮膚常在菌であり、採血時の皮膚消毒が不十分な場合などにコンタミネーションとして検出されやすい菌です。一方で、中心静脈カテーテルや人工物（人工弁・人工関節等）留置中の患者では真の起因菌となることもあるため、患者背景・症状・複数セットの陽性一致状況等を総合して判断する必要があります。
  • c．誤り。CNSはカテーテル関連血流感染や人工物感染など、特定の状況下では真の病原体となり得ます。「常に無視してよい」という判断は誤りです。
  • d．誤り。血液培養は1セットのみでは感度が不十分であり、2セット以上の採取が標準的に推奨されています。複数セット採取は、コンタミネーションと真の菌血症を鑑別するうえでも重要です。
  • e．誤り。セット間で結果が異なること自体は、検査の誤りではなく、コンタミネーションと真の菌血症を鑑別するための重要な情報です。安易に「検査エラー」と結論づけるべきではありません。
整理のポイント：血液培養の解釈は「何が生えたか」だけでなく「何セット中何セットで生えたか」「その菌の典型的な性質（常在菌かどうか）」「患者にカテーテル等の異物があるか」を組み合わせて考えることが基本です。
感染症法・予防接種法

問題5

予防接種法におけるA類疾病・B類疾病に関する記述として、正しいものを1つ選びなさい。

  1. a．B類疾病に分類される定期接種は、集団予防に重点が置かれ、対象者に接種の努力義務が課せられる。
  2. b．季節性インフルエンザおよび高齢者の肺炎球菌感染症に対する定期接種は、B類疾病に位置づけられている。
  3. c．A類疾病に分類される定期接種は「任意接種」に区分され、接種を受けるかどうかは対象者の判断に完全に委ねられている。
  4. d．予防接種法に基づく健康被害救済制度は、A類疾病のみを対象としており、B類疾病の接種による健康被害は救済の対象とならない。
  5. e．新型コロナウイルス感染症の定期接種は、A類疾病として位置づけられ、対象者に接種の努力義務が課せられる。
解答・解説を見る
正解：b
  • a．誤り。「集団予防に重点を置き、努力義務がある」のはA類疾病の説明です。B類疾病は個人の発病・重症化予防に重点が置かれ、対象者への接種の勧奨・努力義務はありません。
  • b．正しい。B類疾病には季節性インフルエンザ、高齢者の肺炎球菌感染症のほか、新型コロナウイルス感染症、帯状疱疹が位置づけられています。いずれも個人予防に重点が置かれ、努力義務は課せられていません。
  • c．誤り。A類疾病は「定期接種」に区分され、対象者（または16歳未満の保護者）には接種の努力義務があります。「任意接種」とは、定期接種の対象年齢外や、そもそも定期接種に含まれない疾病について、本人の希望で受ける接種を指す別の区分です。
  • d．誤り。定期接種による健康被害の救済制度は、A類・B類のいずれの疾病についても対象となります（給付内容には違いがあります）。
  • e．誤り。新型コロナウイルス感染症の定期接種はB類疾病に位置づけられており、努力義務は課せられていません。
整理のポイント：A類＝集団予防・重篤な疾患中心・努力義務あり／B類＝個人予防・季節性インフルエンザや高齢者肺炎球菌等・努力義務なし、という対比で覚えると整理しやすい分野です。
  1. 予防接種法第2条・第9条
