Ollama Chat

ローカルで動作するOllamaのAIモデルをブラウザから手軽に使えるチャットツールです。 外部のAPIキーは不要で、入力したテキストはインターネットに送信されません。 翻訳・要約・校正など、日々の作業をAIにサポートしてもらえます。

⬇ chat.html をダウンロード ▶ デモを開く
🔒

完全ローカル動作

入力内容はすべてお使いのPC内で処理されます。外部サーバーへの送信はありません。

🔤

翻訳ショートカット

日↔英翻訳ボタンをワンクリック。翻訳専門のシステムプロンプトも内蔵しています。

🤖

モデル切り替え

Ollamaで取得済みのモデルを自由に切り替えて使えます。

⚙️

システムプロンプト

AIへの事前指示をUIから設定。「日本語で答えて」などをプリセットから選べます。

動作環境

✓

Ollama のインストール

ollama.com からお使いのOS向けのインストーラをダウンロードしてください。Windows・macOS・Linux に対応しています。

✓

AI モデルの取得

Ollamaインストール後、使いたいモデルをダウンロードします。日本語対応で使いやすいモデル例：

# 軽量で高速（推奨：日本語対応） ollama pull llama3.2 # より高品質（GPUが必要） ollama pull llama3.1:8b
✓

モダンブラウザ

Chrome・Firefox・Edge・Safari の最新版で動作します。Internet Explorerは非対応です。

環境設定

① Ollama を起動する

Ollamaをインストールすると、通常は自動的にバックグラウンドで起動します。手動で起動する場合はターミナルで：

ollama serve

② chat.html をダウンロードして開く

このページ上部のボタンから chat.html をダウンロードし、ファイルをブラウザで直接開いてください（ダブルクリックで開けます）。

③ 接続確認

画面上部に「接続中」と表示されれば準備完了です。接続エラーが出る場合は、OllamaのURLが http://localhost:11434 になっているか確認してください。

CORS 設定（接続できない場合）

ブラウザのセキュリティ制限で接続できない場合は、Ollama 起動時に環境変数を設定してください：

# Windows（コマンドプロンプト） set OLLAMA_ORIGINS=* ollama serve # macOS / Linux OLLAMA_ORIGINS=* ollama serve

基本的な使い方

1

モデルを選ぶ

左サイドバーの「モデル」セレクタから使いたいモデルを選択します。

2

（任意）システムプロンプトを設定する

「必ず日本語で回答」「翻訳専門」などのプリセットを選ぶか、自由に入力できます。何も入れなくてもOKです。

3

メッセージを入力して送信

テキストエリアに入力し、Ctrl + Enter または送信ボタンで送信します。翻訳・要約などのクイックボタンも活用できます。

クイックボタンの使い方

入力エリア上部のボタンをクリックすると、テキストの先頭に定型文が自動入力されます。テキストを貼り付けてからボタンを押しても、先頭に追加されます。
