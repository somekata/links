パート4: 問276-375 | データベース・ネットワーク・クラウド・開発手法

💾 データベース基礎（問276-325）
問276 易
データベースの主な役割は？
正解: B) データを構造的に保存・検索・管理
RDBMS（関係型データベース）が一般的。
問277 易
RDBMS（関係型データベース管理システム）の例は？
正解: B) MySQL, PostgreSQL, Oracle など
テーブル（行・列）で構造化。SQL で操作。
問278 中
SQL の SELECT 文の役割は？
正解: B) テーブルからデータを取得
SELECT col FROM table WHERE condition;
問279 中
SQL の JOIN の用途は？
正解: B) 複数テーブルを条件で結合
INNER JOIN, LEFT JOIN, RIGHT JOIN など種類がある。
問280 中
ノーマライゼーション（正規化）の目的は？
正解: B) 冗長性排除、一貫性保証
第1正規形から第3正規形があり、段階的に改善。
問281 中
主キー（Primary Key）の役割は？
正解: B) 各行を一意に識別
NULL 不可、重複不可。テーブルに 1 つ。
問282 中
トランザクション（Transaction）の特性 ACID の A は？
正解: A) Atomicity（原子性）
トランザクション全体が成功または失敗（中途状態なし）。
問283 中
インデックス（Index）の利点は？
正解: B) 検索速度を向上
B-tree などで高速検索可能。挿入・更新はやや遅くなる。
問284 難
NoSQL データベースの特徴は？
正解: B) スキーマフリー、スケーラビリティ重視
MongoDB（ドキュメント）、Redis（キー値）など。
問285 易
キャッシュの役割は？
正解: B) 頻繁にアクセスするデータを高速に提供
Redis, Memcached などで DB 負荷削減。
🌐 ネットワーク・クラウド（問286-350）
問286 易
LAN（Local Area Network）とは？
正解: B) ローカルエリアネットワーク
イーサネット、Wi-Fi で接続。LAN 内は高速。
問287 易
WAN（Wide Area Network）とは？
正解: B) 広域ネットワーク
インターネット、専用線で異なる地点を接続。
問288 中
ファイアウォール（Firewall）の役割は？
正解: B) 不正な通信を遮断
ハードウェア、ソフトウェアファイアウォール。ポート制御。
問289 中
VPN（Virtual Private Network）の用途は？
正解: B) 安全な通信トンネル
リモートアクセス、データ暗号化で機密性確保。
問290 易
クラウドコンピューティングとは？
正解: B) インターネット上でリソースを利用
AWS, Google Cloud, Azure などで計算・ストレージを提供。
問291 中
IaaS（Infrastructure as a Service）とは？
正解: B) インフラをレンタル
EC2（AWS）などで仮想マシンを提供。ユーザーが OS を管理。
問292 中
PaaS（Platform as a Service）とは？
正解: B) 開発プラットフォームをレンタル
Heroku, Google App Engine でアプリ展開。
問293 易
SaaS（Software as a Service）とは？
正解: B) ソフトウェアを提供
Gmail, Microsoft 365, Salesforce など。インストール不要。
問294 中
コンテナ（Docker）の利点は？
正解: B) 軽量で環境依存性を排除
マイクロサービス、CI/CD で活躍。
問295 中
Kubernetes（K8s）の役割は？
正解: B) コンテナ管理・スケーリング
複数コンテナを自動デプロイ、負荷分散、リカバリ。
🛠️ 開発手法・デプロイ（問296-350）
問296 易
アジャイル開発とウォーターフォール開発の違いは？
正解: B) アジャイル = 短期反復、ウォーターフォール = 全体計画先行
アジャイル = 変更に柔軟、ウォーターフォール = 計画重視。
問297 中
Git のような VCS（Version Control System）の役割は？
正解: B) ソースコード履歴管理
ブランチ、マージで並行開発。GitHub など。
問298 中
CI/CD（Continuous Integration/Continuous Deployment）とは？
正解: B) 変更を自動テスト・デプロイ
Jenkins, GitLab CI で自動化。
問299 易
ユニットテスト（Unit Test）の目的は？
正解: B) 関数単位の動作確認
JUnit, pytest など自動テストツール。
问300 中
リファクタリング（Refactoring）とは？
正解: B) 機能を変えずコード品質向上
可読性、保守性、パフォーマンス改善。
