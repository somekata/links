🔷 C# 応用（問176-225）
問176 易
C# の .NET Framework（または .NET Core）の役割は？
正解: B) ライブラリ、ランタイムを提供
C# は .NET 環境上で動作。CLR（Common Language Runtime）が GC・JIT を管理。
問177 中
C# の LINQ（Language Integrated Query）の用途は？
正解: B) SQL 的なクエリで操作
from x in list where x.Age > 20 select x.Name のように直感的。
問178 中
C# の async/await の機能は？
正解: B) 非同期処理を同期的に記述
await で待機、UI ブロッキングなし。スレッドプールで効率実行。
問179 中
C# のプロパティ（Property）と フィールド（Field）の違いは？
正解: B) get/set でカプセル化
プロパティで値の検証・変換が可能（public int Age { get; set; }）。
問180 中
C# のイベント（Event）の役割は？
正解: B) 疎結合な通知機構
button.Click += EventHandler のようにデリゲート登録。
問181 中
C# のデリゲート（Delegate）とは？
正解: B) 型安全な関数ポインタ
delegate int MyDelegate(int x); で型を定義、変数に関数を割り当て。
問182 難
C# の null合体演算子（??）の用途は？
正解: B) null の場合に代替値を指定
var x = value ?? defaultValue; で null チェック。
問183 中
C# のタプル（Tuple）の利点は？
正解: B) 複数値をまとめて返す
(int age, string name) = GetPersonInfo(); で簡潔。
問184 易
C# の var キーワードの役割は？
正解: B) 型推論（コンパイル時に型決定）
var x = 5; で int と推論される。
問185 中
C# の extension method（拡張メソッド）とは？
正解: B) 既存クラスにメソッドを追加
public static bool IsEven(this int n) { } で int に IsEven を追加。
問186 中
C# の interface と abstract class の使い分けは？
正解: B) interface = 契約、abstract = 共通実装
複数 interface 実装 OK、abstract は単一継承（C# 8.0 以降は実装可能）。
🌐 Web 技術基礎（問187-225）
問187 易
HTTP リクエストの主なメソッドは？
正解: A) GET, POST, PUT, DELETE など
GET = 取得、POST = 送信、PUT = 更新、DELETE = 削除。
問188 易
HTTP ステータスコード 200 の意味は？
正解: B) 成功
2xx = 成功、3xx = リダイレクト、4xx = クライアントエラー、5xx = サーバーエラー。
問189 易
HTTP ステータスコード 404 の意味は？
正解: B) Not Found（ファイル見つからない）
404 エラーはクライアント側の error（4xx）。
問190 中
REST API の設計原則として正しいものは？
正解: A) ステートレス、URL でリソース指定
/api/users/123 GET = ユーザー取得。HTTP メソッドで操作指定。
問191 中
JSON（JavaScript Object Notation）の役割は？
正解: B) 軽量なデータ交換形式
{ "name": "Taro", "age": 30 } のような構造。XML より簡潔。
問192 中
CORS（Cross-Origin Resource Sharing）の目的は？
正解: B) 異なるオリジン間の安全な通信
ブラウザの SOP を回避し、サーバーが同意した場合に通信許可。
問193 中
Cookie と Session の違いは？
正解: B) Cookie = クライアント、Session = サーバー
Cookie は改ざん可能だが高速、Session は安全だがサーバー負荷。
問194 中
OAuth 2.0 の用途は？
正解: B) サードパーティアプリへの安全な認可
「Google でログイン」は OAuth 2.0。パスワード共有なし。
問195 中
JWT（JSON Web Token）の特徴は？
正解: B) ステートレス、署名付き情報
Header.Payload.Signature で構成。改ざん検知可能。
問196 易
HTML の役割は？
正解: B) Web ページの構造定義
HTML = 骨組み、CSS = 装飾、JavaScript = 動作。
問197 易
CSS の役割は？
正解: B) デザイン・スタイル指定
色、フォント、レイアウトなどを指定。
問198 易
JavaScript の役割は？
正解: C) インタラクティブな動作・ロジック
ボタンクリック、フォーム検証、AJAX 通信など。
問199 中
SQL インジェクション攻撃とは？
正解: B) SQL コマンド混入で DB 乗っ取り
対策：プリペアドステートメント、入力値検証。
問200 中
XSS（Cross-Site Scripting）攻撃とは？
正解: B) 悪意のある JavaScript 注入
Cookie 盗聴、フィッシング誘導など。対策：入力エスケープ。
