ローカルAI翻訳 / Review Translator

査読コメントを、
安全に翻訳する。

Ollamaを使ったローカル動作の翻訳ツールです。テキストは外部に送信されず、すべて手元のPCで処理されます。日本語 ⇄ 英語の双方向に対応。

review_translator.html をダウンロード

※ ダウンロード後、ローカルでダブルクリックして開いてください。

完全ローカル動作 データ外部送信なし 日本語 ⇄ 英語 追加インストール不要（本ツール）

セットアップ手順

1
Ollamaをインストールする
ollama.com からWindows版インストーラーをダウンロードして実行します。インストール後、PCを再起動してください。
2
翻訳モデルをダウンロードする（初回のみ）
PowerShellを開いて、以下のコマンドでモデルを取得します。数分〜十数分かかります。
Windows PowerShell
PS C:\Users\username> ollama pull gemma3
pulling manifest...
pulling ffe83c... 100% ████████████ 3.3 GB
success

他のモデルは下の「モデル一覧」を参照。追加する場合も同様に ollama pull モデル名 で取得できます。

3
ツールをダウンロードする
このページ上部のボタンから review_translator.html をダウンロードし、任意のフォルダに保存します。
4
Ollamaを起動する（使用前に毎回）
PowerShellでツールのあるフォルダに移動し、Ollamaを起動します。このウィンドウは翻訳中も開いたままにしてください。
Windows PowerShell
PS C:\Users\username> cd Documents\review_translator
PS C:\Users\username\Documents\review_translator> $env:OLLAMA_ORIGINS="*"; ollama serve
Listening on 127.0.0.1:11434 ← これが出れば起動成功
（このウィンドウは閉じないでください）

cd の後のパスは、review_translator.html を保存した場所に合わせて変更してください。

5
ツールを開く
review_translator.html をダブルクリックしてブラウザで開きます。右上のステータスが 「接続OK」 になっていれば準備完了です。

モデル一覧

ツール起動時にインストール済みのモデルが自動で表示されます。複数入れておくと用途に応じて切り替えられます。

モデルの特徴

モデル名 サイズ 特徴・用途
gemma3推奨 約 3.3 GB バランスが良く高精度。日常的な査読コメントに最適。
llama3 約 4.7 GB 英語表現が自然。英語→日本語翻訳向け。
mistral 約 4 GB 速度と品質のバランスが良い。大量翻訳作業に。
qwen2.5 約 4 GB アジア言語に強い。専門的な技術用語を含む文に。

RAMに合わせた推奨モデル

翻訳速度はお使いのPCのRAMに大きく依存します。下記を参考にモデルを選んでください。

RAM 推奨モデル 備考
8 GB以下 gemma3 重いモデルは動作しない場合があります。gemma3のみを推奨。
16 GB gemma3 / mistral 快適に動作します。用途に合わせて選択してください。
32 GB以上 すべてのモデル llama3・qwen2.5など大きめのモデルも余裕をもって動作します。

新しいモデルを追加するには別のPowerShellウィンドウで ollama pull モデル名 を実行してください。

使い方の早見表

毎回の起動

  • 1PowerShellを開く
  • 2ツールのフォルダへ移動
  • 3$env:OLLAMA_ORIGINS="*"; ollama serve を実行（ウィンドウは閉じない）
  • 4HTMLをダブルクリックで開く
  • 5右上が「接続OK」になっていることを確認

翻訳の手順

  • 1「⇄」ボタンで翻訳の方向を選ぶ
  • 2左の枠に査読コメントを貼り付ける
  • 3「翻訳する →」または Ctrl+Enter
  • 4右側の結果を「📋 コピー」でコピー

終了方法

  • 1ブラウザのタブを閉じる
  • 2PowerShellで Ctrl+C を押す
  • 3PowerShellウィンドウを閉じる

ショートカット

  • Ctrl+Enter　翻訳を実行
  • Ctrl+C　Ollamaを停止（PowerShell内）

トラブルシューティング

「Ollama未接続」と表示される
PowerShellでOllamaが起動しているか確認。起動していない場合は起動コマンドを再実行してください。
「bind: Only one usage...」エラーが出る
Ollamaがすでに起動中です。Shift+Ctrl+Esc でタスクマネージャーを開き、ollama.exe をすべて終了してから再起動してください。
翻訳ボタンを押しても何も起きない
別のPowerShellウィンドウで ollama list を実行し、モデルが表示されるか確認してください。表示されない場合は ollama pull gemma3 でインストールしてください。
翻訳が非常に遅い
PCのメモリが不足している可能性があります。gemma3（小さいモデル）を選択するか、他のアプリを閉じてください。推奨RAM：16 GB以上。
「ollama」コマンドが認識されない
Ollamaが正しくインストールされていません。PCを再起動後、ollama.com から再インストールしてください。
