00 / BASICS — まず言葉を理解しよう
OllamaってなんでLlamaと関係あるの？

「AI」「LLM」「Llama」「Ollama」……似たような言葉が並んで混乱しますよね。3つを分けて考えると一気にクリアになります。

🧠
LLM
Large Language Model（大規模言語モデル）
ChatGPTやClaudeの「頭脳」部分のことです。膨大な文章を学習して、次に来る言葉を予測するAIの仕組みです。

「7B」「13B」などの数字はパラメータ数（Billion＝10億）を意味します。数が大きいほど賢いですが、その分重くなります。
💡 例え：LLMは「すごく頭のいい脳みそ」。それ単体では動けないので、動かすための「体」が必要です。
🦙
Llama（ラマ）
Meta社が開発した無料公開のAIモデル
Facebook・Instagramを作ったMeta社（旧Facebook）が開発・無料公開した高性能なAIモデルです。名前の由来は南米の動物のラマ（🦙）。

通常AIは有料サービスとして提供されますが、Llamaは設計図ごと無料公開（オープンソース）されたため、世界中の開発者が自由に使えます。
💡 例え：Llamaは「無料で配布されている高性能エンジン」。このエンジンを使えば誰でも車（アプリ）を作れます。
⚙️
Ollama（オラマ）
ローカルAIを動かすための管理ソフト
LlamaなどのAIモデルを自分のPCで簡単に動かすためのソフトウェアです。名前は「Ol（古い）」＋「Llama」から来ています。

本来、AIを動かすには複雑な設定が必要でした。Ollamaはその面倒な部分をすべて引き受けてくれます。コマンド1行でAIをダウンロード・起動できます。
💡 例え：OllamaはAIモデルの「DVDプレイヤー」。Llama（DVD）をセットして再生ボタン（コマンド）を押すだけで動きます。

02 / SETUP — セットアップ手順
インストールから動かすまで

4ステップで完了します。コマンドはコピー＆ペーストするだけでOKです。

STEP 01
// INSTALL OLLAMA
Ollamaをインストールする
AIの「DVDプレイヤー」となるソフトウェアを導入します。難しい設定は一切ありません。
1
ollama.com からWindows版をダウンロードして実行します。インストーラーが全部やってくれます。
2
スタートメニューで powershell と検索して、PowerShell（黒い画面）を起動します。
3
下のコマンドを貼り付けてEnterキーを押します。バージョン番号が表示されれば成功です。
PowerShell
SHELL
ollama --version
# → ollama version 0.x.x  と表示されれば成功！
STEP 02
// DOWNLOAD MODEL
AIモデルをダウンロードする
AIの「脳」を取得します。最初は gemma3 がおすすめです（日本語が得意）。
PowerShell
SHELL
# ① モデルをダウンロード（2〜5GB。Wi-Fiで数分かかります）
ollama pull gemma3

# ② ダウンロード完了後、チャットで動作確認
ollama run gemma3

# ③ 画像認識もやりたい場合はこちらも追加
ollama pull llava
💡
ダウンロードに時間がかかるのは正常です。一度ダウンロードすれば次回からは即座に起動します。データはすべて自分のPC内に保存され、外部には一切送信されません。
STEP 03
// BUILD YOUR APP
Node.jsで専用AIアプリを作る
プログラムからAIを操作し、「論文査読専用AI」など特定業務に特化した自分専用ツールに変えます。
1
nodejs.org から推奨版（LTS）をインストールします。これもインストーラーが自動でやります。
2
デスクトップに my-ai フォルダを作り、PowerShellで cd ~/Desktop/my-ai と入力して移動します。
3
npm install ollama を実行してライブラリをインストールします。
4
下のコードをメモ帳に貼り付けて app.js として保存し、node app.js で実行します。
app.js — 論文査読AI（Gemma 3版）
JAVASCRIPT
const ollama = require('ollama');

async function reviewPaper() {
  const response = await ollama.default.chat({
    model: 'gemma3',   // ← ここのモデル名を変えるだけでAIの「脳」が切り替わります
    messages: [
      {
        role: 'system',   // ← AIへの「役割」の指示（毎回自動で送られる）
        content: 'あなたは優秀な医学論文の査読者です。'
                + '客観的かつ建設的な英語コメントを生成してください。'
      },
      {
        role: 'user',
        content: '以下のアブストラクトを査読してください：\n[ここに論文を貼る]'
      }
    ],
  });

  console.log(response.message.content);
}

reviewPaper();
