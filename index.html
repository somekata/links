PlayGround

オンラインでコーディングを練習するためのサイトです。

  • https://colab.research.google.com/: 厳密にはPlayGroundではないが練習には使える Pythonが基本だがRも使える
  • https://codapi.org/: Rのコードを含む多言語対応 Runがやや遅いかも
  • https://playcode.io/new: HTML CSS JS Python等に対応 Rはない 即時反映
  • https://developer.mozilla.org/en-US/play: JSに対応 HTMLとCSSも対応 Mozilla Foundationが運営している公式サイト
  • https://codepen.io/pen/: HTML CSS JSの練習
  • https://www.jsplayground.dev/: JSのみの対応 HTMLとCSSはない 余計な機能が不要な場合には便利
  • https://jsfiddle.net/: HTML CSS JSの練習
  • https://programiz.pro/ide/python: Python C C++ Java Goに対応 https://programiz.pro/ide/rでRにも対応
  • https://python-playground.netlify.app/: Pythonのみ対応
  • https://www.boot.dev/playground/py: Python Go JavaScript TypeScript SQL C Luaに対応 Rはない HTMLとCSSもない Runは比較的速い

Rの場合、GoogleColabでランタイム➨ランタイムのタイプを変更➨ランタイムのタイプ➨Python3からRに変更　することで使える。
ただし、pythonと違って、初めからライブラリがインストールされていないので、都度インストールする必要がある。
codapiやProgramizでは一部の機能が制限されており、ライブラリがインストールできない。どちらも、描画機能は使えない。
codapiでは、write.csvやread.csvも制限されているが、dplyrは使えるのでパイプ演算子は使える。
Programizでは、write.csvやread.csvは可能だが、dplyrが使えない。
