会話で学ぶ：
ボールを投げるプログラム

先生と学生の会話を追いながら、
「ボタンを押すとボールが飛ぶ」プログラムを段階的に作っていきます。

01

まずはアイデアを決めよう

いまから、プログラミングについて説明します。 例えば、ボタンを押すと何かが起こる、というプログラムを考えてみましょう。
何が起こるようにするの？
いい質問だね。じゃあ、ボタンを押したらボールが飛んでいくことにしよう。 もちろん本当に飛ぶわけではなくて、画面の上で動く、という意味だよ。
02

何を作る必要がある？

さて、最初に何を作る？
ボール？ ボタン？
そうだね。どちらも必要だね。 じゃあ、まずはボールから考えよう。ボールは丸だね。 このボールに名前を付けたいので、id="ball" のように名前を付ける考え方を使います。
ボタンにも名前がいる？
その通り。何を押したのか分かるように、ボタンにも名前が必要だね。 今回は id="throw" にしよう。
03

動きを作るには？

次に必要なのは、「押したら動く」という処理だね。
この処理を function で作ります。 function は、「こういう仕事をしてください」という命令のまとまりだと思えばいいよ。
じゃあ、投げる処理を作ればいいんだね。
そう。今回は throwBall という名前にしよう。 そして、ボタンが押されたことを聞き取るために、イベントリスナーを使うよ。 「押した」という出来事を見張ってくれる仕組みなんだ。
04

まずは HTML を書こう

HTMLは、画面に何を置くかを書くものです。今回は、ボールを描く場所として canvas と、押すための button を置きます。

HTML の構造
<body>
  <h1>ボタンを押すとボールが飛ぶ</h1>
  <canvas id="canvas" width="500" height="180"></canvas>
  <button id="throw">投げる</button>
</body>
💡 canvas とは？
以前の説明では <div id="ball"> としていましたが、実際に丸を描いて動かすなら canvas を使うほうが自然です。 canvas はJavaScriptでグラフィックを描画できる「キャンバス（画布）」です。
05

JavaScript の考え方

次は JavaScript だよ。ここが「動き」の本体になる。
まず、canvas を取り出して、そこに丸を描けるように準備する。
それから、ボールの位置を表す x と y を決める。
そして、drawBall() でボールを描く。
最後に、throwBall() で少しずつ右へ動かして、「飛んでいるように見せる」んだ。
JavaScript のコード（解説付き）
// ① canvas と button を取得
const canvas      = document.getElementById("canvas");
const ctx         = canvas.getContext("2d");     // 描画の道具箱
const throwButton = document.getElementById("throw");

// ② ボールの初期位置
let x = 40;
let y = 90;

// ③ ボールを描く関数
function drawBall() {
  ctx.clearRect(0, 0, canvas.width, canvas.height); // 一度消す
  ctx.beginPath();
  ctx.arc(x, y, 15, 0, Math.PI * 2);               // 丸を描く
  ctx.fillStyle = "tomato";
  ctx.fill();
}

// ④ 投げる関数（setInterval で繰り返し動かす）
function throwBall() {
  let count = 0;

  const timer = setInterval(() => {
    x += 10;          // 右に10pxずつ進む
    drawBall();
    count++;

    if (count >= 25) {
      clearInterval(timer); // 25回動いたら止まる
    }
  }, 200);            // 0.2秒ごとに実行
}

// ⑤ ボタンが押されたら throwBall を呼ぶ
throwButton.addEventListener("click", throwBall);

// ⑥ 最初にボールを表示しておく
drawBall();
💡 setInterval とは？
setInterval(処理, ミリ秒) は「指定した時間ごとに処理を繰り返す」タイマーです。 今回は 200ms（0.2秒）ごとにボールを右へ10px動かし、25回繰り返すことで「飛んでいる」ように見せています。
06

実際に動かしてみよう

ボタンを押すと、ボールが右へ飛んでいきます。

ライブデモ

ボタンを押すごとにボールが飛びます。飛んでいる途中は押せません。

07

完成した HTML + JS の全コード

これをそのまま throwball_final.html として保存すれば、単体で動きます。

throwball_final.html — 全コード
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>throwball</title>
  <style>
    body   { font-family: sans-serif; text-align: center; padding: 30px; }
    canvas { border: 1px solid #ccc; border-radius: 8px;
             display: block; margin: 20px auto; }
    button { font-size: 18px; padding: 10px 20px; border: none;
             background: #10b981; color: white;
             border-radius: 8px; cursor: pointer; }
  </style>
</head>
<body>
  <h1>ボタンを押すとボールが飛ぶ</h1>
  <canvas id="canvas" width="500" height="180"></canvas>
  <button id="throw">投げる</button>

  <script>
    const canvas      = document.getElementById("canvas");
    const ctx         = canvas.getContext("2d");
    const throwButton = document.getElementById("throw");

    let x = 40, y = 90, moving = false;

    function drawBall() {
      ctx.clearRect(0, 0, canvas.width, canvas.height);
      ctx.beginPath();
      ctx.arc(x, y, 15, 0, Math.PI * 2);
      ctx.fillStyle = "tomato";
      ctx.fill();
    }

    function throwBall() {
      if (moving) return;   // 動いている間は無視
      moving = true;
      x = 40;
      drawBall();
      let count = 0;

      const timer = setInterval(() => {
        x += 10;
        drawBall();
        count++;
        if (count >= 25) { clearInterval(timer); moving = false; }
      }, 200);
    }

    throwButton.addEventListener("click", throwBall);
    drawBall();
  </script>
</body>
</html>
