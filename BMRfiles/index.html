01

最初に知っておく用語

パソコン操作やプログラミングでよく出てくる言葉です。「なんとなく知っている」ではなく、きちんと説明できるレベルにしておきましょう。

ファイル
データをひとまとまりにして保存したもの。文書・画像・プログラムなど、あらゆるデータがファイルとして保存される。
ディレクトリ
（= フォルダ）
ファイルをまとめて入れる「入れ物」。プログラムでは「フォルダ」より「ディレクトリ」と呼ぶことが多い。
拡張子
ファイル名の末尾についている .html .css .js などの文字列。ファイルの種類を示す。
インストール
ソフトウェアをパソコンに導入して使えるようにすること。アプリを「入れる」こと。
クリック
マウスの左ボタンを1回押すこと。ファイルやボタンを選択・実行するときに使う。
右クリック
マウスの右ボタンを押すこと。コンテキストメニュー（操作メニュー）が表示される。
ダブルクリック
マウスの左ボタンを素早く2回押すこと。ファイルを開いたりアプリを起動するときに使う。
ドラッグ
マウスのボタンを押したまま動かすこと。ファイルの移動や範囲選択に使う。
ドラッグ＆ドロップ
ドラッグして目的の場所でボタンを離すこと。ファイルの移動・コピーに使う。
コピペ
（コピー＆ペースト）
コピーして貼り付けること。Ctrl+C → Ctrl+V（Mac は ⌘）。
パス
ファイルやフォルダの場所を示す文字列。例：C:\Users\taro\index.html
ブラウザ
Webページを表示するソフト。Chrome・Edge・Safari・Firefoxなど。HTMLを開く場所でもある。
⚠ 拡張子が見えていない場合（Windows）：
エクスプローラー → 「表示」タブ → 「ファイル名拡張子」にチェックを入れて表示させましょう。

VSCode の準備

拡張機能 Prettier が使える

VSCode（Visual Studio Code）は、Microsoftが無料で提供するコードエディタです。プログラミング学習から実務まで広く使われています。

↗ VSCode 公式サイト（ダウンロード）
1
VSCode をダウンロード・インストール
上のリンクから自分のOS（Windows / Mac）に合ったものをダウンロードし、指示に従ってインストールする。
2
拡張機能パネルを開く
左サイドバーの四角いアイコン（Extensions）をクリック、または Ctrl+Shift+X（Mac: ⌘+Shift+X）を押す。
3
Live Server をインストール
検索欄に Live Server と入力 → 「Ritwick Dey」作のものの「Install」をクリック。

使い方：HTMLファイルを右クリック → 「Open with Live Server」でブラウザに自動表示。保存するたびに自動更新される。
4
Prettier をインストール
検索欄に Prettier と入力 → 「Prettier - Code formatter」の「Install」をクリック。

使い方：ファイルを右クリック → 「Format Document」でコードが自動整形される。
💡 Live Server とは：
保存するたびにブラウザが自動で更新されます。コードを編集 → 保存 → ブラウザが即反映、という流れが快適になります。講義でも必ず使います。

04

HTML の基本

HTMLは Webページの「骨格（構造）」を作る言語です。「何を書くか」を決めます。

HTML = タグの集まり
<タグ名> と </タグ名> のペアで内容を囲みます。このペアを要素と呼びます。
基本構造
<!DOCTYPE html>
<html>
  <head>
    <title>ページのタイトル</title>
  </head>
  <body>
    <h1>見出し</h1>
    <p>段落のテキスト</p>
    <button>ボタン</button>
  </body>
</html>
<html>
HTMLの始まりと終わりを示すタグ。すべてのHTMLはこの中に書く。
<head>
ブラウザに表示されない情報を書く場所。タイトルや外部ファイルの読み込みなど。
<body>
ブラウザに表示されるコンテンツを書く場所。見出し・文章・ボタンなどここに書く。
<h1> 〜 <h6>
見出しタグ。数字が小さいほど大きな見出し。h1が最大、h6が最小。
<p>
段落（paragraph）。ひとまとまりの文章を囲む。
<div>
意味を持たない「箱」タグ。要素をグループ化するときによく使う。
<button>
クリックできるボタン。JSと組み合わせることで動きを付けられる。
id / class 属性
タグに名前をつける属性。CSSやJSから特定の要素を指定するときに使う。
⚠ よくある間違い：タグは必ずペアで閉じましょう。<p> を書いたら </p> を忘れずに。
05

CSS の基本

CSSは Webページの「見た目（デザイン）」を指定する言語です。「どう見せるか」を決めます。

CSS の書き方： セレクタ { プロパティ: 値; } の形で書きます。
基本の書き方
/* タグ名で指定 */
p {
  color: red;          /* 文字色 */
  font-size: 18px;     /* 文字サイズ */
}

/* id で指定（# をつける） */
#myTitle {
  font-size: 24px;
  text-align: center;
}

/* class で指定（. をつける） */
.box {
  background-color: skyblue;
  padding: 10px;       /* 内側の余白 */
  margin: 10px;        /* 外側の余白 */
}
セレクタ
「どの要素に」スタイルを適用するかを指定する部分。タグ名・id・classなどで指定する。
プロパティ
color や font-size など、変更したいスタイルの種類。コロン（:）の左側。
値（バリュー）
red や 18px など、プロパティに設定する具体的な内容。コロン（:）の右側。
padding
要素の内側の余白。背景色はpaddingの部分まで広がる。
margin
要素の外側の余白。隣の要素との間隔を調整する。
/* コメント */
実行されないメモ書き。コードに説明を加えるときに使う。
06

JavaScript の基本

JavaScriptは Webページに「動き（インタラクション）」を加えるプログラミング言語です。HTML・CSSと合わせて3つで1セットです。

HTML — 構造
<button id="btn">
  クリック
</button>
<p id="msg"></p>
CSS — 見た目
button {
  background: navy;
  color: white;
  padding: 8px 16px;
}
JS — 動き
btn.addEventListener(
  'click',
  function() {
    msg.textContent =
      'こんにちは！';
  }
);
JavaScript の最重要概念
// ① 変数（データを入れる箱）
const name = '太郎';    // 変えない値には const
let score = 0;          // 変わる値には let

// ② 関数（処理をまとめたもの）
function greet() {
  console.log('こんにちは、' + name);
}
greet(); // → こんにちは、太郎

// ③ 条件分岐（もし〜なら）
if (score >= 60) {
  console.log('合格');
} else {
  console.log('不合格');
}

// ④ 繰り返し（〜を○回くり返す）
for (let i = 0; i < 3; i++) {
  console.log(i); // 0, 1, 2
}
💡 console.log( ) とは：
ブラウザの「開発者ツール」（F12 キーで開く）のコンソール画面に値を表示する命令です。プログラムの動きを確認したり、バグを探すときに使います。
変数
データを入れておく「名前のついた箱」。const（変更不可）と let（変更可）を使い分ける。
関数
一連の処理をひとまとめにして名前をつけたもの。function 名前() { } で定義し、名前() で呼び出す。
条件分岐
if / else を使い、条件によって処理を変える。「もし〜なら」の仕組み。
繰り返し
for や while を使い、同じ処理を何度もくり返す。
DOM
JSからHTMLの要素を操作する仕組み。document.getElementById('id名') で要素を取得できる。
イベント
クリック・キー入力などユーザーの操作のこと。addEventListener で「〜されたら処理する」と登録する。
