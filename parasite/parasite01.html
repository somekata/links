00なぜ寄生虫を学ぶのか

教員
今日から寄生虫学に入ります。「寄生虫」と聞いて、どんなイメージを持ちますか。
学生
あまり馴染みがなく、少し怖いイメージです。昔の病気というか、上下水道が整っていなかった時代の問題というイメージがあります。今の日本ではもうあまり関係ないのかな、と。
教員
その印象は半分正しく、半分は誤解です。衛生インフラの向上で回虫・鉤虫のような腸管寄生虫は日本国内で激減しました。しかし寄生虫学が「過去の学問」かというと、まったくそうではありません。
寄生虫学を学ぶ意義

① グローバル化と輸入感染症　海外渡航者・在留外国人の増加により、マラリアや住血吸虫症などが臨床現場に現れます。発熱患者の鑑別診断において渡航歴の聴取が不可欠なのはこのためです。

② 食習慣に根ざした国内感染症　魚介類や野生動物の生食文化が続く日本では、アニサキス症・肺吸虫症など独自の寄生虫感染症が維持されています。

③ 免疫不全患者の増加　移植医療・HIV感染・生物学的製剤の普及により、健常者では問題にならない日和見寄生虫（クリプトスポリジウムなど）が重症化しうる患者が増えています。

教員
もう一つ、寄生虫学が面白いのは、「目に見える生物としての生活環」を持つ点です。卵から幼虫・成虫へと姿を変え、複数の宿主を渡り歩く生活環の理解が、感染経路と予防策の理解に直結します。暗記科目ではなく、生物の生活史を追う物語として捉えてください。

01寄生虫学とは何か

1-1. 寄生(parasitism)の定義

教員
「寄生」を生物学的に定義してみましょう。
学生
他の生物に乗っかって栄養を奪う関係でしょうか。
教員
おおむね正しいです。厳密には「異種の生物間で、一方（寄生体）が他方（宿主）から一方的に利益を受け、宿主は通常不利益を被る関係」です。「共生（symbiosis）」という大きな概念の中のひとつの形態にあたります。
関係性一方の生物他方の生物
相利共生 (mutualism)利益利益
片利共生 (commensalism)利益影響なし
寄生 (parasitism)利益不利益

ただし、病害を起こさない「腸管内常在原虫」のように片利共生に近い関係を示す種も存在し、寄生か共生かは宿主の免疫状態や虫体量によって連続的に変化します。

1-2. 医学寄生虫学が扱う範囲

医学寄生虫学（medical parasitology）は、狭義には原虫（protozoa）と蠕虫（helminths）を扱いますが、広義には媒介節足動物（衛生動物学）も含みます。本シリーズでは主に原虫・蠕虫を扱い、節足動物は媒介者（ベクター）として必要な範囲で言及します。

1-3. 細菌・ウイルスとの対比

項目ウイルス細菌原虫蠕虫
細胞構造非細胞性原核単細胞真核単細胞真核多細胞
増殖の場細胞内のみ細胞内外細胞内外主に細胞外
宿主体内での増殖ありありあり原則なし
診断の基本核酸検査・抗原培養・グラム染色鏡検（原虫体）鏡検（虫卵・幼虫）
重要な原則

蠕虫は宿主体内で原則として数を増やしません。「虫体量 ＝ もともとの感染虫数」であり、再感染を繰り返さない限り自然に虫体数は減っていきます。これは原虫・ウイルス・細菌の体内増殖とは根本的に異なる病態理解の前提です。

02原虫と蠕虫の違い

教員
原虫（Protozoa）と蠕虫（Helminths）の違いを、サイズだけでなく構造の観点から説明できますか。
学生
原虫は単細胞で小さく、蠕虫は多細胞で大きいということでしょうか。
教員
その通りです。単細胞の原虫は一つの細胞に運動・摂食・排泄のすべての機能を持たせる必要があります。多細胞の蠕虫は消化管・生殖器官・体壁といった分化した器官系を持てます。このサイズと構造の違いが、診断法（鏡検の検体・染色法）や免疫応答（Th1型 vs Th2型）の違いに直結します。

2-1. 原虫の分類

原虫は伝統的に運動様式で4群に分類されます。

分類群運動小器官代表例
根足虫類 (Sarcodina/Amoebozoa)仮足 (pseudopodium)Entamoeba histolytica（赤痢アメーバ）
鞭毛虫類 (Mastigophora/Flagellata)鞭毛 (flagellum)Giardia lamblia、Trypanosoma 属、Leishmania 属
胞子虫類 (Sporozoa/Apicomplexa)運動小器官を欠く（特殊滑走運動）Plasmodium 属、Toxoplasma gondii、Babesia 属

2-2. 蠕虫の分類

形態による覚え方

線虫は「丸くて筒状」、吸虫・条虫は扁形動物門に属し「平たい」。さらに吸虫は「分節なし」、条虫は「片節（proglottid）が連なる」という形で区別できます。

03宿主の機能的分類

教員
宿主の分類を整理します。まず「機能的分類」として、宿主の体内で寄生虫が何をするかという観点から3種に分けます。

3-1. 終宿主 (definitive host)

寄生虫が成虫になる、あるいは有性生殖を行う宿主を終宿主と呼びます。これは機能的・段階的な定義です。マラリアでは有性生殖（オーシスト形成）がハマダラカ体内で起こるため、蚊が終宿主、ヒトは中間宿主という、直感に反する分類になります。

3-2. 中間宿主 (intermediate host)

寄生虫の幼虫期の発育・無性生殖が行われる宿主です。複数介在する場合は感染順に第一・第二中間宿主と呼びます。

虫卵
→
第一中間宿主
（巻貝など）
→
第二中間宿主
（魚・カニなど）
→
終宿主
（ヒトなど）

3-3. 待機宿主 (paratenic host)

幼虫がそれ以上発育せず、ただ生存し続けるだけの宿主です。終宿主への「橋渡し」の役割を果たします。アニサキスにおける各種魚類（待機宿主的な役割）が典型例です。

宿主の種類定義具体例（寄生虫/宿主）
終宿主成虫・有性生殖の場有鉤条虫/ヒト、マラリア原虫/ハマダラカ
中間宿主幼虫期・無性生殖の場肝吸虫/巻貝（第一）、コイ科淡水魚（第二）
待機宿主発育せず生存・伝播のみアニサキス/魚類
⚠️ 日本の教科書における用語の注意

日本の寄生虫学教科書の多くでは、終宿主を英語の "final host" に、固有宿主を "definitive host" に対応させ、両者を同義として扱う記述が見られます。この表記では「definitive host ＝ 固有宿主（本来のライフサイクルにおける宿主）」という生態学的概念として用いられています。

一方、国際的な標準的用法（WHOガイドラインや英語の寄生虫学教科書）では definitive host は「有性生殖が行われる宿主」すなわち本テキストで「終宿主（functional sense）」と呼ぶものに相当します。

英語文献・日本語教科書を参照する際は、その文献がどの定義で用語を使っているかを文脈から確認することが重要です。本テキストでは国際的用法に準拠し、終宿主＝definitive host（機能的定義）、固有宿主＝natural host（生態学的定義）として区別します。

Q. マラリアにおいて、ヒトと蚊のどちらが終宿主でしょうか。
蚊（ハマダラカ）が終宿主です。蚊体内で配偶子が受精しオーシストを経てスポロゾイトが形成される有性生殖が起こるためです。ヒト体内では無性生殖（裂殖）のみが行われるため、ヒトは中間宿主に分類されます。

04宿主の生態学的分類と2軸マトリクス

教員
第3章では「何をするか」という機能的分類を学びました。しかし実際の寄生虫学には、もう一つ重要な軸があります。それは「本来のライフサイクルに含まれる宿主かどうか」という生態学的・進化的な軸です。
学生
機能的分類と生態学的分類、何が違うのですか？
教員
終宿主・中間宿主という機能的分類は「その宿主体内で寄生虫が有性生殖するか幼虫期を過ごすか」という役割の話です。一方、生態学的分類は「その宿主が自然界における本来のライフサイクルに組み込まれているか否か」という話です。この2つは独立した軸で、組み合わせることでより精密に宿主の位置づけを記述できます。

4-1. 固有宿主 (natural host)

寄生虫が自然界での本来のライフサイクルにおいて利用する宿主種を指します。生態学的・進化的な概念であり、「固有終宿主（natural definitive host）」「固有中間宿主（natural intermediate host）」のように、機能的分類と組み合わせて使用されます。固有宿主においては宿主と寄生虫の間で長い共進化が起こっているため、病害が比較的軽微で「均衡した共生関係」に近い状態にあることが多いです。

「固有宿主」と「definitive host」の用語対応について

日本の教科書では definitive host を「固有宿主」と訳し、final host を「終宿主」に対応させる場合があります（両者を同義として扱うものも多い）。この場合の definitive host は「本来のライフサイクルにおける宿主」という生態学的意味で使われています。一方、本テキストおよび多くの英語文献では definitive host ＝「有性生殖が行われる宿主」（機能的定義の終宿主）として使います。英語原文を読む際は文脈に応じてどちらの意味で使われているかを確認してください。

4-2. 偶発宿主 (accidental host / aberrant host)

本来のライフサイクルに含まれない宿主種に、偶発的に感染が成立してしまう場合を指します。偶発宿主では共進化が起こっていないため、宿主側に過剰な病害が生じやすく（例：幼虫移行症）、寄生虫側は発育不全・成虫化不能に終わることが多いです。機能的分類との組み合わせで「偶発終宿主」「偶発中間宿主」「偶発待機宿主」のように記述できます。

4-3. 代替宿主 (alternative host)

固有宿主ほど最適ではないものの、固有宿主が不在の場合にライフサイクルを実質的に維持できる宿主種です。固有宿主よりも感染効率や寄生虫の生存率が低いことが多いですが、宿主の行動域・個体数などの生態学的条件によっては重要な疫学的役割を果たすことがあります。

4-4. 保虫宿主 (reservoir host)

公衆衛生上・疫学上の重要概念で、ヒトに感染する寄生虫が自然界で絶滅せずに維持されている「源（リザーバー）」となる動物を指します。多くの場合その動物にとっては「固有宿主」であり、長い共進化の結果として宿主側にほとんど症状が出ない安定した状態にあります。保虫宿主の同定は、感染症対策（人獣共通感染症のサーベイランス、ベクターコントロールなど）において核心的な情報となります。

2軸マトリクスの読み方

「機能軸」と「生態軸」は独立した分類体系であり、どちらの軸も中間宿主・終宿主・待機宿主のいずれにも適用できます。従来の「異常宿主」という曖昧な用語は、このマトリクス上では「偶発宿主」と「機能的位置づけ」の組み合わせ（偶発中間宿主・偶発待機宿主など）として、より正確に記述できます。

05生活環の基本用語

ミラシジウム (miracidium)
虫卵から水中で孵化する繊毛を持つ幼生。遊泳し第一中間宿主（主に巻貝）に侵入する。ギリシャ語 meirakidion（若者）に由来。
スポロシスト (sporocyst)
貝内でミラシジウムが脱繊毛・変態した袋状構造。内部で無性生殖を繰り返しレジアを産生する。
レジア (redia)
原始的な消化管を持つ幼生。貝の組織内でさらに無性生殖を行い、セルカリアを多数産生する。
セルカリア (cercaria)
尾部を持ち自由遊泳能力を持つ幼生。貝から脱出し第二中間宿主へ侵入する。ギリシャ語 kerkos（尾）に由来。
メタセルカリア (metacercaria)
セルカリアが第二中間宿主の体内・表面で脱尾・被嚢した休止期幼生。終宿主への感染形態で、生食によりヒトに感染する病原体の実体はこの段階であることが多い。
例外：住血吸虫

Schistosoma 属（住血吸虫）はセルカリアがヒト皮膚を直接経皮貫通して感染するため、第二中間宿主・メタセルカリア期を経ません。

5-2. 条虫類の発育段階

六鉤幼虫 (oncosphere)
虫卵内に存在する6本の鉤を持つ幼虫。中間宿主に摂取されると孵化し腸壁から血流に入り組織へ移行する。
嚢尾虫 (cysticercus)
六鉤幼虫が中間宿主の組織内で発育し頭節を内包した嚢状になった幼虫。有鉤条虫における「豚肉のはしか」として知られる。
包虫 (hydatid cyst)
エキノコックス属に特有の幼虫嚢胞。多数の原頭節を内部に形成し緩徐に増大する。

5-3. 線虫類の発育段階

線虫類は脱皮（molting）によりL1〜L5の幼虫期を経て成虫に至ります。吸虫・条虫のような「異なる形態名称を持つ変態」ではなく、同じ基本形態のまま脱皮を重ねる発育様式が特徴です。

ステージ内容
L1〜L2環境中または中間宿主体内での幼虫期
L3（感染幼虫）多くの線虫で終宿主への感染力を獲得する重要なステージ
L4〜L5終宿主体内での発育段階。L5を経て成虫（adult）となる

5-4. 原虫の発育段階

原虫は主に栄養型（trophozoite）とシスト型（cyst）の2形態を行き来します。複数宿主を介する胞子虫類（マラリア原虫・トキソプラズマ）では、スポロゾイト・メロゾイト・配偶子（ガメトサイト）・オーシストなど、より細分化された用語体系が用いられます。

用語体系の比較

吸虫・条虫は段階ごとに固有の名称を持つ変態を行う／線虫は同じ基本形態のまま脱皮を重ねる／原虫は栄養型・シスト型という二形性を基本とする、という3つの体系の違いを押さえると各論学習が整理しやすくなります。

06感染経路の分類

教員
感染経路は生活環のどの段階でヒトに接触するかによって決まります。4つに分類できます。

07寄生虫と免疫・病態の基礎

学生
寄生虫感染症では好酸球が増えるとよく聞きますが、なぜですか。
教員
蠕虫感染に対する宿主の主要な防御機構は、ウイルス・細菌に対するものとは異なり、2型ヘルパーT細胞（Th2）優位の免疫応答です。Th2細胞が産生するIL-5は骨髄での好酸球産生を強く刺激し、IgE産生もこの経路で誘導されます。
蠕虫感染の臨床的特徴

① 末梢血好酸球増多（特に組織侵入期に顕著）　② 血清IgE高値　③ 慢性的・遷延性の経過（成虫が長期生存し急性炎症よりも慢性炎症・線維化が主体）。これらは原虫感染・細菌感染の病態とは対照的です。

一方、原虫感染では病原体ごとに免疫応答のパターンが大きく異なります。細胞内寄生性の原虫（トキソプラズマ、リーシュマニアなど）に対しては細胞性免疫（Th1優位・マクロファージ活性化）が主要な防御機構となるため、細胞性免疫不全（臓器移植後・HIV感染・ステロイド長期使用など）の患者で重症化・日和見感染化しやすいという特徴があります。

発展：なぜ寄生虫は宿主免疫から逃れられるのか

多くの寄生虫は長期間生存するための免疫回避機構を備えています。抗原変異（トリパノソーマの表面抗原スイッチング）、分子擬態（宿主分子の模倣）、免疫抑制性サイトカイン誘導、被嚢構造による物理的隔離（包虫嚢胞の外膜など）が代表的な機序です。これらの機序はワクチン開発が細菌・ウイルスに比べて困難な理由の説明にもつながります。

G用語集（総論編）

終宿主 (definitive host)
成虫・有性生殖が行われる宿主。機能的定義
中間宿主 (intermediate host)
幼虫期・無性生殖が行われる宿主。機能的定義
待機宿主 (paratenic host)
発育せず生存・伝播のみを担う宿主。機能的定義
固有宿主 (natural host)
本来のライフサイクルに組み込まれている宿主。生態学的定義。終宿主・中間宿主・待機宿主のいずれにも適用可能
偶発宿主 (accidental host)
本来のライフサイクル外で偶発的に感染が成立する宿主。生態学的定義。共進化が起こっていないため病害が過剰になりやすい
代替宿主 (alternative host)
固有宿主ほど最適ではないが、固有宿主不在時にライフサイクルを維持できる宿主
保虫宿主 (reservoir host)
ヒトに感染する寄生虫が自然界で維持される源となる動物。多くの場合その動物にとっては固有宿主であり病害が軽微
栄養型 (trophozoite)
原虫の活動的な増殖形態
シスト (cyst)
原虫の環境抵抗性休止形態
ミラシジウム (miracidium)

セルカリア (cercaria)

メタセルカリア (metacercaria)
セルカリアが被嚢した終宿主への感染形態
六鉤幼虫 (oncosphere)
条虫の鉤を持つ虫卵内幼虫
嚢尾虫 (cysticercus)
条虫の中間宿主組織内幼虫

病原体を媒介する節足動物
